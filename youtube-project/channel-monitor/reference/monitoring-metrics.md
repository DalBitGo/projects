# YouTube 채널 모니터링 지표 상세

## 1. 핵심 성과 지표 (Primary KPIs)

### Watch Time (시청 시간)
- **정의**: 시청자가 영상을 시청한 총 시간
- **중요성**: YouTube 알고리즘의 가장 중요한 랭킹 요소
- **측정 방법**: YouTube Analytics API - `watchTime` 메트릭
- **활용**:
  - 채널별 총 시청 시간 비교
  - 영상별 시청 시간 분석
  - 시간대별 시청 패턴 파악

### Audience Retention (시청 유지율)
- **정의**: 영상의 각 시점에서 남아있는 시청자 비율
- **측정 단위**: 퍼센트 (%)
- **분석 포인트**:
  - Average Percentage Viewed: 평균 시청 비율
  - Audience Retention Curve: 시점별 이탈 그래프
  - Key Moments: 이탈률 높은 구간
- **활용**:
  - 콘텐츠 구성 최적화
  - 인트로/아웃트로 효과성 검증
  - 콘텐츠 길이 결정

### CTR (Click-Through Rate)
- **정의**: 노출 대비 클릭 비율
- **계산식**: (클릭 수 / 노출 수) × 100
- **중요성**: 썸네일과 제목의 효과성 직접 측정
- **벤치마크**:
  - 2-10%: 일반적인 범위
  - 10% 이상: 매우 우수
- **개선 방향**:
  - A/B 테스트로 썸네일 최적화
  - 제목 키워드 최적화

## 2. 성장 지표 (Growth Metrics)

### 조회수 (Views)
- **추적 항목**:
  - 총 조회수
  - 일일 조회수 변화
  - 업로드 후 경과 시간별 조회수
  - 트래픽 소스별 조회수
- **분석 패턴**:
  - 첫 24시간 조회수 (초기 반응)
  - 첫 주 조회수 (바이럴 가능성)
  - 장기 조회수 (SEO 효과)

### 구독자 수 (Subscribers)
- **추적 항목**:
  - 총 구독자 수
  - 일일 구독자 증감
  - 영상별 구독자 전환율
  - 구독 취소율
- **중요 지표**:
  - Net Subscriber Change: 순 구독자 변화
  - Subscriber Conversion Rate: (구독자 증가 / 조회수) × 100

### 참여율 (Engagement Rate)
- **구성 요소**:
  - 좋아요 (Likes)
  - 싫어요 (Dislikes) - 현재 비공개
  - 댓글 (Comments)
  - 공유 (Shares)
  - 저장/재생목록 추가
- **계산식**: (참여 행동 총합 / 조회수) × 100
- **벤치마크**: 3-6% 일반적

## 3. 트래픽 소스 분석

### 주요 트래픽 소스
1. **YouTube 검색**: SEO 효과
2. **추천 영상**: 알고리즘 선호도
3. **외부 소스**: SNS, 블로그 유입
4. **채널 페이지**: 기존 구독자
5. **재생목록**: 콘텐츠 시리즈 효과
6. **알림**: 구독자 충성도

### 분석 활용
- 트래픽 소스별 성과 비교
- 마케팅 채널 ROI 측정
- 콘텐츠 발견 경로 최적화

## 4. 시청자 분석 (Audience Analytics)

### 인구통계
- 연령대 분포
- 성별 분포
- 지역/국가 분포
- 언어 선호도

### 시청 패턴
- 활동 시간대 (업로드 최적 시간)
- 기기 유형 (모바일/데스크톱/TV)
- 재방문율
- 신규 vs 재방문 시청자 비율

## 5. 수익 지표 (Monetization Metrics)

### RPM (Revenue Per Mille)
- 1,000회 조회당 수익
- 채널별 수익성 비교

### CPM (Cost Per Mille)
- 1,000회 광고 노출당 수익
- 광고주 입찰가 지표

### 예상 수익
- 일일/월간 수익
- 영상별 수익 기여도

## 6. 콘텐츠 성과 분석

### 콘텐츠 타입별 분류
- 주제/카테고리별 성과
- 영상 길이별 성과
- 시리즈/단독 영상 성과
- 포맷별 성과 (튜토리얼, 브이로그, 리뷰 등)

### 비교 분석
- 채널 간 동일 콘텐츠 타입 비교
- 시간대별 성과 변화
- 경쟁 채널 벤치마크
