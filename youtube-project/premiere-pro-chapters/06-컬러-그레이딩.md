### Chapter 6: 컬러 그레이딩

#### 6.1 컬러 그레이딩이란?

**정의**
- 영상의 색감, 밝기, 대비 조정
- 특정 무드/분위기 연출
- 시네마틱한 느낌 만들기

**컬러 보정 vs 컬러 그레이딩**
- **Color Correction**: 문제 해결 (너무 어두움, 색온도 이상 등)
- **Color Grading**: 창의적 표현 (영화 같은 느낌, 빈티지, 따뜻함 등)

#### 6.2 Lumetri Color 패널 이해하기

**패널 열기**
- Window → Lumetri Color
- 또는 작업 공간을 "Color"로 변경

**Lumetri Color 섹션 구조**
1. **Basic Correction**: 기본 색 보정
2. **Creative**: 창의적 Look 적용
3. **Curves**: 정밀 톤 조정
4. **Color Wheels & Match**: 색상 휠 조정
5. **HSL Secondary**: 특정 색상만 조정
6. **Vignette**: 비네팅 효과

#### 6.3 Basic Correction (기본 보정)

**주요 슬라이더**

**1. Input LUT**
- 카메라 Log 파일 변환
- 일반 촬영에서는 사용 안 함

**2. White Balance (화이트 밸런스)**
- **Temperature**: 색온도 조정
  - 좌측 (파란색): 차갑게
  - 우측 (노란색): 따뜻하게
- **Tint**: 마젠타/그린 조정
  - 좌측: 그린 (형광등)
  - 우측: 마젠타 (자연스러운 피부톤)

**3. Tone (톤 조정)** ⭐ 가장 중요
- **Exposure**: 전체 밝기 (-2 ~ +2)
- **Contrast**: 대비 (어두운 부분↓, 밝은 부분↑)
- **Highlights**: 밝은 영역만 조정
- **Shadows**: 어두운 영역만 조정
- **Whites**: 가장 밝은 부분
- **Blacks**: 가장 어두운 부분

**4. Saturation (채도)**
- 색의 선명도 (-100 ~ +100)
- 너무 높으면 부자연스러움 (±20 이내 권장)

**일반적인 조정 순서**
1. Exposure로 전체 밝기 맞춤
2. Contrast로 선명도 증가 (+10~20)
3. Highlights 낮춤 (-10~-20) → 밝은 부분 디테일 살림
4. Shadows 올림 (+10~+20) → 어두운 부분 밝게
5. Saturation 살짝 증가 (+5~10)

#### 6.4 Creative (크리에이티브 Look)

**Look 프리셋**
- Premiere Pro 내장 LUT(Look Up Table)
- 영화 느낌, 빈티지, 시네마틱 등
- Look 드롭다운에서 선택

**인기 Look**
- **Teal and Orange**: 영화 같은 느낌
- **Cinematic**: 시네마틱 톤
- **Vintage**: 빈티지 필름 느낌
- **Black & White**: 흑백

**Adjustments**
- **Faded Film**: 빈티지 페이드 효과 (0-100)
- **Sharpen**: 선명도 (10-30)
- **Vibrance**: 자연스러운 채도 증가
- **Saturation**: 전체 채도

**팁**: Look 적용 후 Intensity로 강도 조절 (50-80% 권장)

#### 6.5 Color Wheels & Match (색상 휠)

**3-Way Color Corrector 개념**
- **Shadows**: 어두운 부분 색조 조정
- **Midtones**: 중간톤 색조 조정
- **Highlights**: 밝은 부분 색조 조정

**사용 방법**
- 색상 휠 중앙의 점을 드래그
- 따뜻하게: 노란색/오렌지 방향
- 차갑게: 파란색/시안 방향
- 미세 조정으로 분위기 연출

**시네마틱 룩 만들기 (Teal & Orange)**
1. Shadows → 약간 시안(Teal) 방향
2. Highlights → 약간 오렌지 방향
3. 대비 효과로 영화 같은 느낌

#### 6.6 Curves (커브 조정)

**RGB Curve**
- 가장 정밀한 톤 조정 도구
- S-Curve로 대비 증가

**S-Curve 만들기**
1. 곡선 중앙 상단 클릭 → 위로 드래그 (밝은 부분 더 밝게)
2. 곡선 중앙 하단 클릭 → 아래로 드래그 (어두운 부분 더 어둡게)
3. "S" 모양 형성 → 대비 증가, 선명도 향상

**Hue Saturation Curves**
- 특정 색상만 채도 조정
- 예: 하늘(파란색)만 더 선명하게

#### 6.7 실전 컬러 그레이딩 워크플로우

**시나리오**: 평범한 영상을 시네마틱하게

**1단계: Basic Correction (2분)**
- Exposure: 전체 밝기 확인
- Contrast: +15
- Highlights: -15
- Shadows: +10
- Saturation: +5

**2단계: Creative Look 적용 (1분)**
- Look → "Teal and Orange" 선택
- Intensity: 70%

**3단계: S-Curve 적용 (1분)**
- Curves 섹션 열기
- RGB 커브에서 S-Curve 생성
- 미세 조정

**4단계: Color Wheels (선택, 1분)**
- Shadows → 약간 Teal
- Highlights → 약간 Orange
- 강도 조절

**총 소요 시간: 약 5분**

#### 6.8 프리셋 저장 및 재사용

**프리셋 저장**
1. Lumetri Color 조정 완료
2. Effects → Presets → Create Preset
3. 이름 입력 (예: "My Cinematic Look")
4. Save

**프리셋 적용**
1. Effects → Presets → My Presets
2. 클립에 드래그 앤 드롭
3. 일관된 색감 유지

#### 6.9 컬러 매칭 (Color Match)

**시나리오**: 여러 클립의 색감 통일

**자동 매칭**
1. 기준 클립 (잘 나온 색감) 지정
2. 조정할 클립 선택
3. Lumetri Color → Color Wheels & Match → Comparison View
4. Apply Match
5. 자동으로 유사한 색감 적용

#### 6.11 자주 하는 실수와 해결법

**실수 1: 과도한 채도로 부자연스러운 색감**
- **원인**: Saturation을 +30 이상 높게 설정
- **증상**: 피부톤이 주황색으로 변하고, 하늘이 형광색처럼 보임
- **해결**:
  - Saturation을 -20 ~ +15 범위 내로 제한
  - Vibrance 사용 권장 (피부톤 보호하며 채도 증가)
  - HSL Secondary로 특정 색상만 선택적 조정
- **프로 기준**: 일반 영상은 ±10, 뮤직비디오는 +15~20
- **복구**: Saturation을 0으로 리셋 후 Vibrance +10부터 시작

**실수 2: Highlights를 너무 낮춰 디테일 손실**
- **원인**: Highlights -50 이하로 설정하여 밝은 영역이 회색으로 변함
- **증상**: 하늘이 흐리고, 밝은 부분 디테일 사라짐
- **해결**:
  - Highlights는 -30 이내로 제한
  - Whites와 조합하여 사용
  - Curves에서 하이라이트 영역 세밀 조정
- **권장 설정**:
  - Highlights: -10 ~ -20 (적당한 디테일 보존)
  - Whites: -5 ~ -10 (가장 밝은 부분만 조절)

**실수 3: 어두운 영상을 밝힐 때 노이즈 증가**
- **원인**: Exposure를 +1.0 이상 올려 원래 숨어있던 노이즈가 드러남
- **해결**:
  - Exposure 대신 Shadows와 Blacks를 우선 조정
  - Denoise 이펙트 적용: Effects → Noise Reduction & Grain → Denoise
  - Amount: 5-10 (살짝만 적용)
  - 또는 Neat Video 플러그인 사용 (유료)
- **촬영 팁**: ISO를 낮게 설정하여 촬영 시 노이즈 방지

**실수 4: 색온도 조정 실패로 부자연스러운 피부톤**
- **원인**: Temperature를 극단적으로 조정 (±50 이상)
- **증상**: 피부가 너무 파랗거나 노랗게 보임
- **해결**:
  - White Balance Selector (스포이드) 사용
  - 흰색 또는 회색 물체 클릭하여 자동 보정
  - Tint와 함께 조정 (마젠타/그린 밸런스)
- **수동 조정 가이드**:
  - Temperature: -20 ~ +20 (미세 조정)
  - Tint: +5 ~ +15 (마젠타 쪽, 자연스러운 피부톤)

**실수 5: 모든 클립에 같은 프리셋 적용**
- **원인**: 조명 조건이 다른 클립에 동일한 Lumetri 프리셋 사용
- **증상**: 일부 클립은 너무 밝고, 일부는 너무 어두움
- **해결**:
  - 프리셋 적용 후 개별 클립마다 Basic Correction 재조정
  - Exposure, Highlights, Shadows를 클립별로 미세 조정
  - Color Match 기능으로 클립 간 색감 통일
- **워크플로우**: 프리셋 적용 → 개별 조정 (5분 소요)

**실수 6: S-Curve를 너무 극단적으로 만듦**
- **원인**: 커브의 상단/하단을 과도하게 드래그
- **증상**: 대비가 너무 강해 눈이 피로하고, 색이 깨짐
- **해결**:
  - S-Curve는 부드럽게 (완만한 S자)
  - 상단 포인트: +10~15 정도만 올림
  - 하단 포인트: -10~15 정도만 내림
  - 키프레임 우클릭 → Bezier로 부드러운 곡선
- **비교**: 원본과 수시로 비교 (이펙트 on/off 토글)

**실수 7: Vignette를 과도하게 적용**
- **원인**: Vignette Amount -50 이하로 설정
- **증상**: 화면 가장자리가 터널처럼 보임, 부자연스러움
- **해결**:
  - Vignette Amount: -10 ~ -20 (미묘하게)
  - Midpoint: 50-60 (비네팅 범위 조절)
  - Feather: 70-80 (부드러운 전환)
- **사용 시기**: 시네마틱 장면, 피사체 강조 시에만 사용

#### 6.12 프로 팁

**팁 1: Lumetri Scopes로 과학적 색보정**
- Window → Lumetri Scopes 열기
- **Waveform (RGB)**: 밝기 분포 확인
  - 상단(100): 가장 밝은 부분
  - 하단(0): 가장 어두운 부분
  - 고른 분포가 이상적
- **Vectorscope YUV**: 색상 분포 확인
  - 중앙: 무채색
  - 바깥쪽: 채도 높은 색
  - 피부톤 라인(Skin Tone Line)에 피부색이 위치해야 자연스러움
- **활용**: 밝기 클리핑 방지, 색상 편향 확인
- **시간 절약**: 육안보다 정확하고 빠름

**팁 2: LUT 적용 후 강도 조절**
- LUT(Look Up Table)는 강력하지만 과할 수 있음
- Basic Correction → Input LUT → LUT 선택
- Creative → Look 대신 Input LUT 사용 시:
  - 조정 레이어 사용 (Adjustment Layer)
  - 레이어 불투명도로 LUT 강도 조절 (50-80%)
- **프로 워크플로우**:
  1. LUT 적용 (80% 강도)
  2. Basic Correction으로 밝기 조정
  3. Creative에서 미세 조정
- **LUT 소스**: 무료 - RocketStock, IWLTBAP / 유료 - FilmConvert, Dehancer

**팁 3: 조정 레이어로 여러 클립 동시 그레이딩**
- 여러 클립에 같은 색보정 일괄 적용
- **방법**:
  1. Project 패널 → New Item → Adjustment Layer
  2. 타임라인의 최상위 비디오 트랙(V3, V4)에 배치
  3. Adjustment Layer에 Lumetri Color 적용
  4. 아래 모든 클립에 동일하게 영향
- **장점**: 수정 시 한 번만 조정하면 모든 클립 변경
- **시간 절약**: 10개 클립 개별 조정(30분) → 조정 레이어(5분)

**팁 4: Color Wheels로 영화적 색분리**
- 할리우드 영화의 Teal & Orange Look 재현
- **방법**:
  1. Color Wheels & Match 섹션
  2. Shadows: 시안(Teal) 방향으로 살짝 이동
  3. Midtones: 중립 유지
  4. Highlights: 오렌지 방향으로 살짝 이동
  5. 각 휠 10-20% 정도만 이동 (미묘하게)
- **효과**: 차가운 그림자, 따뜻한 하이라이트로 입체감과 깊이 증가
- **참고 영화**: "Mad Max: Fury Road", "Blade Runner 2049"

**팁 5: HSL Secondary로 특정 색상만 조정**
- 하늘만 더 파랗게, 잔디만 더 초록색으로
- **워크플로우**:
  1. HSL Secondary 섹션 펼치기
  2. Eyedropper로 원하는 색상 선택 (예: 하늘 클릭)
  3. Key → Color/Hue/Sat/Lum 범위 조정
  4. Correction → Saturation, Temperature 조정
- **실전 예시**:
  - 하늘: Saturation +20, Temperature -10 (더 파랗게)
  - 피부톤: Saturation -5, Highlights +10 (자연스럽게)
  - 식물: Saturation +15 (생동감)
- **마스킹**: Denoise로 선택 범위 부드럽게

**팁 6: 시간대별 색감 프리셋 구축**
- 자주 쓰는 시간대별로 프리셋 저장
- **프리셋 예시**:
  - "Golden Hour Warm": 일몰 영상용 (Temperature +15, Saturation +10)
  - "Blue Hour Cool": 밤 영상용 (Temperature -20, Tint +10)
  - "Midday Bright": 낮 영상용 (Exposure +0.3, Contrast +20)
  - "Overcast Muted": 흐린 날 (Saturation -10, Highlights +10)
- **저장 위치**: Effects → Presets → My Presets
- **효율**: 매번 처음부터 시작 안 해도 됨

**팁 7: 전/후 비교 모드로 정확한 판단**
- 그레이딩 중 원본과 수시로 비교
- **방법**:
  1. 프로그램 모니터 하단 비교 버튼 클릭
  2. 또는 Lumetri Color 패널 상단 전원 아이콘 on/off
  3. 단축키 설정: Keyboard Shortcuts → "Enable Lumetri" 할당 (예: Shift + E)
- **분할 화면 비교**:
  - Comparison View → Side-by-Side
  - Reference Frame 설정하여 before/after 동시 확인
- **팁**: 5분 작업 후 반드시 원본 비교 (눈의 적응 방지)

#### 6.13 실전 예시: 영화 같은 Look 만들기

**시나리오**: 평범한 여행 브이로그를 영화 같은 시네마틱 Look으로 변환

**준비 단계 (1분)**
1. 작업 공간을 "Color"로 변경 (상단 Window → Workspaces → Color)
2. Lumetri Color 패널과 Lumetri Scopes 모두 열기
3. 조정할 클립 선택

**1단계: 색온도 및 기본 노출 보정 (3분)**
- White Balance 조정:
  1. 영상이 너무 차가우면 (푸른 기운) Temperature +10
  2. 형광등 느낌이면 Tint +5 (마젠타 추가)
- 기본 노출:
  1. Waveform 보며 전체 밝기 확인
  2. Exposure: +0.2 ~ +0.5 (살짝 밝게)
  3. Lumetri Scopes에서 파형이 0-100 사이 고르게 분포 확인
- **결과**: 중립적이고 균형 잡힌 출발점

**2단계: 대비 및 톤 조정 (5분)**
- 시네마틱 대비 생성:
  1. Contrast: +20 (선명도 증가)
  2. Highlights: -20 (밝은 부분 디테일 살림)
  3. Shadows: +15 (어두운 부분 약간 밝혀 디테일 보존)
  4. Whites: -10 (가장 밝은 부분 클리핑 방지)
  5. Blacks: -5 (깊이 있는 검은색)
- **Waveform 확인**:
  - 상단이 100에 닿지 않게 (클리핑 방지)
  - 하단이 0에 살짝 닿거나 가까이 (진한 블랙)
- **효과**: 영화 같은 풍부한 톤 범위

**3단계: S-Curve로 대비 강화 (4분)**
- Curves 섹션:
  1. RGB 커브 선택
  2. 중앙 상단 클릭 → 위로 드래그 (+15 정도)
  3. 중앙 하단 클릭 → 아래로 드래그 (-15 정도)
  4. 부드러운 S자 곡선 생성
- **미세 조정**:
  - 커브가 너무 급격하지 않게
  - 각 포인트 우클릭 → Bezier로 부드러운 전환
- **효과**: 밝은 부분은 더 밝게, 어두운 부분은 더 어둡게 → 입체감

**4단계: Teal & Orange Look 적용 (6분)**
- Creative 섹션:
  1. Look → "Teal and Orange" 선택
  2. Intensity: 60-70% (자연스럽게)
  3. Adjustments:
     - Faded Film: 10-15 (필름 느낌)
     - Sharpen: 15-20 (선명도)
     - Vibrance: +10 (자연스러운 채도)
- Color Wheels로 미세 조정:
  1. Shadows → 시안(Teal) 쪽으로 10-15 이동
  2. Highlights → 오렌지 쪽으로 10-15 이동
  3. Midtones → 거의 중립 유지
- **효과**: 차가운 그림자 + 따뜻한 하이라이트 = 영화적 색분리

**5단계: HSL Secondary로 하늘 강조 (4분)**
- 하늘을 더 드라마틱하게:
  1. HSL Secondary 섹션 펼치기
  2. Eyedropper로 하늘의 파란색 클릭
  3. Key 설정:
     - Hue: 파란색 범위 확장 (180-240도)
     - Saturation: 중간~높음 범위
     - Lum: 밝은 영역 (하늘)
  4. Correction:
     - Temperature: -10 (더 차갑게)
     - Saturation: +15 (더 선명하게)
     - Exposure: +0.1 (살짝 밝게)
- **마스킹 확인**: Key → Output → Highlight로 선택 영역 확인 (하늘만 선택되었는지)

**6단계: Vignette로 시선 유도 (2분)**
- 화면 중앙 강조:
  1. Vignette 섹션
  2. Amount: -15 (가장자리 어둡게)
  3. Midpoint: 55 (비네팅 범위)
  4. Roundness: 0 (자연스러운 모양)
  5. Feather: 75 (부드러운 전환)
- **효과**: 시청자 시선이 중앙 피사체로 집중

**7단계: 프리셋 저장 및 적용 (3분)**
- 현재 설정을 프리셋으로 저장:
  1. Effects 패널 → Presets
  2. 우클릭 → Create Preset
  3. 이름: "Cinematic Travel Vlog"
  4. Save
- 다른 클립에 적용:
  1. 프리셋을 다른 클립에 드래그
  2. 각 클립마다 Exposure와 Highlights만 미세 조정 (조명 차이)
  3. 5-10개 클립에 1-2분 내 적용 가능

**8단계: 최종 확인 및 내보내기 (3분)**
- 전체 재생하며 확인:
  - 클립 간 색감 일관성
  - 피부톤 자연스러움 (주황색 아님)
  - 클리핑 없음 (Waveform 확인)
- 다양한 기기에서 테스트:
  - 컴퓨터 모니터
  - 스마트폰
  - TV
- 필요 시 Master Adjustment Layer로 최종 미세 조정

**총 소요 시간: 약 30분 (첫 클립 완성 후 나머지는 10분)**

**Before vs After 비교**
- Before: 평범한 노출, 생기 없는 색감, 플랫한 대비
- After: 풍부한 톤, 시네마틱 색분리, 입체감, 영화 같은 분위기

#### 6.14 고급 컬러 기법

**기법 1: 색상 격리 및 교체**
- 특정 색상을 완전히 다른 색으로 변경
- **워크플로우**:
  1. HSL Secondary → Eyedropper로 대상 색상 선택 (예: 빨간 셔츠)
  2. Key 범위 조정하여 해당 색상만 격리
  3. Correction → Hue 슬라이더로 색상 변경 (빨강 → 파랑)
  4. Saturation과 Luminance도 함께 조정
- **활용 사례**:
  - 브랜드 컬러 통일 (로고, 의상)
  - 배경 색상 변경 (초록 벽 → 파랑)
  - 계절 분위기 변경 (초록 나무 → 주황/노랑 단풍)

**기법 2: 색상별 Curves 조정**
- RGB가 아닌 Hue vs Sat, Hue vs Hue, Hue vs Lum 커브 활용
- **Hue vs Saturation Curve**:
  1. Curves → Hue Saturation Curves 선택
  2. 특정 색상 범위(예: 빨강) 포인트 추가
  3. 위로: 해당 색만 채도 증가
  4. 아래로: 채도 감소
- **Hue vs Hue Curve**:
  - 색상 자체를 부드럽게 변경
  - 예: 주황색을 빨강 쪽으로 이동
- **활용**: 피부톤 미세 조정, 자연 색상 강조
- **프로 팁**: 일반 S-Curve보다 정밀한 제어 가능

**기법 3: 영화 필름 룩 재현 (Film Emulation)**
- 디지털 영상을 필름 카메라처럼 보이게
- **설정**:
  1. Creative → Look: "Kodak 2383" 또는 "Fuji 3510"
  2. Curves → RGB Curve 하단 약간 올리기 (블랙을 회색으로 = Lifted Blacks)
  3. Saturation: -5 ~ -10 (약간 탈색)
  4. Sharpen: -5 (부드러운 느낌)
  5. Video Effects → Noise & Grain → Film Grain 적용 (Amount: 10-20)
- **효과**: 빈티지 영화 느낌, 따뜻하고 노스탤직한 분위기
- **참고 영화**: "Her", "Carol", "The Grand Budapest Hotel"

**기법 4: Day for Night (낮을 밤으로)**
- 낮에 촬영한 영상을 밤처럼 보이게 (예산 절감)
- **워크플로우**:
  1. Basic Correction:
     - Exposure: -1.0 ~ -1.5 (어둡게)
     - Contrast: +25 (대비 증가)
     - Shadows: -20 (그림자 더 어둡게)
  2. Creative:
     - Temperature: -30 (차갑고 푸른 밤 느낌)
     - Tint: +10 (마젠타 추가로 자연스럽게)
  3. Color Wheels:
     - 모든 톤에 파란색 추가
  4. HSL Secondary:
     - 하늘 선택 → 어둡게 + 파랗게
- **주의**: 완벽한 재현은 어려움, 달빛 느낌 정도
- **보완**: 음향 효과로 밤 분위기 강화

**기법 5: 복수 Lumetri 이펙트 레이어링**
- 하나의 클립에 여러 Lumetri Color 이펙트 적용
- **방법**:
  1. 클립에 Lumetri Color 적용 (기본 보정)
  2. Effects 패널에서 Lumetri Color 한 번 더 드래그 (크리에이티브)
  3. Effects 패널에서 또 한 번 드래그 (특수 효과)
- **레이어 구조 예시**:
  - Lumetri 1: Basic Correction (노출, 대비)
  - Lumetri 2: Creative Look (Teal & Orange)
  - Lumetri 3: HSL Secondary (하늘 보정)
  - Lumetri 4: Vignette 및 마무리
- **장점**: 각 단계 독립적으로 on/off 가능, 디버깅 쉬움
- **주의**: 너무 많으면 (4개 이상) 렌더링 느려짐

**기법 6: 색공간 및 HDR 워크플로우**
- 고급 색공간 관리로 전문가 수준 결과물
- **Log 촬영 워크플로우**:
  1. 카메라 Log 파일 (S-Log, Log-C 등) 임포트
  2. Basic Correction → Input LUT → 카메라 LUT 선택 (Rec.709 변환)
  3. 이후 일반 그레이딩 진행
- **HDR (High Dynamic Range)**:
  1. 시퀀스 설정 → Color Space: Rec. 2020 또는 HDR
  2. Lumetri Scopes → HDR Waveform 확인
  3. 더 넓은 색영역과 밝기 범위 활용
  4. 내보내기 시 HDR 메타데이터 포함
- **활용**: Netflix, YouTube HDR 콘텐츠, 극장 상영
- **장비**: HDR 모니터 필요 (정확한 모니터링)

**기법 7: 색보정 참조 이미지 활용**
- 영화 스틸컷을 참조로 정확한 색감 재현
- **방법**:
  1. 원하는 영화의 스크린샷 (고품질 PNG) 임포트
  2. 타임라인 V2에 참조 이미지 배치
  3. 불투명도 30-50%로 낮춤 (반투명)
  4. V1의 실제 클립 위에 오버레이
  5. 참조 이미지와 매칭되도록 Lumetri 조정
  6. 완료 후 참조 이미지 숨기기
- **팁**: Comparison View로 나란히 비교도 가능
- **효과**: 추상적 감각 대신 구체적 목표로 정확한 결과

#### 6.15 문제 해결

**문제 1: 색보정 후 피부톤이 부자연스러움 (주황색/초록색)**
- **원인**: Saturation이나 Temperature 과도 조정, 또는 Look의 강도가 너무 높음
- **해결**:
  1. Vectorscope YUV에서 피부톤 확인
  2. 피부톤 라인(Skin Tone Line)에 위치해야 정상
  3. HSL Secondary로 피부톤만 격리
  4. Correction → Saturation -5 ~ -10 (채도 낮춤)
  5. Temperature +5 (약간 따뜻하게)
- **예방**: 피부가 있는 장면은 Vibrance 사용 (Saturation 대신)
- **참조**: 백인 피부톤 벡터 위치: 11시 방향, 아시아인: 11-12시 방향

**문제 2: 클립 간 색감이 불일치**
- **증상**: 연속된 장면인데 색감이 갑자기 바뀜 (점프컷 느낌)
- **원인**: 각 클립마다 다른 프리셋 또는 개별 조정
- **해결**:
  1. 가장 잘 나온 클립을 "마스터" 클립으로 선택
  2. 다른 클립 선택 → Lumetri Color → Color Match
  3. Comparison View에서 마스터 클립 지정
  4. Apply Match → 자동 매칭
  5. 필요 시 Exposure만 미세 조정
- **또는**:
  - 조정 레이어 사용하여 모든 클립에 동일 그레이딩
  - 개별 차이는 Basic Correction만 조정

**문제 3: Lumetri Color 적용 시 재생이 느려짐 (렉)**
- **원인**: 고해상도 (4K) + 복수 Lumetri 이펙트로 실시간 처리 과부하
- **해결**:
  1. 프로그램 모니터 → 재생 해상도: 1/4 또는 1/2로 낮춤
  2. Sequence → Render In to Out (Enter) - 해당 구간 렌더링
  3. 또는 프록시 워크플로우 사용
  4. GPU 가속 확인: Project Settings → Mercury Playback Engine GPU
- **최종 확인**: 내보내기 전에만 Full Resolution로 확인
- **사양 업그레이드**: GPU (NVIDIA RTX 시리즈 권장)

**문제 4: 색보정이 내보낸 파일에 적용 안 됨**
- **증상**: 편집 화면에서는 보이는데 Export 후 원본 색감
- **원인**:
  - Lumetri Color 이펙트가 비활성화됨 (fx 아이콘 꺼짐)
  - 조정 레이어가 숨겨짐 또는 음소거
  - Rendering 설정 문제
- **해결**:
  1. Effect Controls에서 Lumetri Color fx 아이콘 확인 (켜져 있어야 함)
  2. 조정 레이어의 눈 아이콘 확인 (켜져 있어야 함)
  3. Sequence Settings → Maximum Bit Depth 체크
  4. Export Settings → Use Maximum Render Quality 체크
- **테스트**: 짧은 구간 (5초)만 먼저 내보내서 확인

**문제 5: HDR 영상이 색이 이상하게 보임**
- **증상**: HDR 촬영 영상이 회색빛으로 보이거나 색이 탈색됨
- **원인**: Log 색공간 파일을 Rec.709로 변환 안 함
- **해결**:
  1. Basic Correction → Input LUT
  2. 카메라 제조사 LUT 선택:
     - Sony: S-Log3 to Rec.709
     - Canon: C-Log to Rec.709
     - Panasonic: V-Log to Rec.709
  3. LUT 적용 후 일반 그레이딩 진행
- **또는**: Sequence Settings에서 Working Color Space 변경

**문제 6: Vignette가 타원형으로 왜곡됨**
- **원인**: 시퀀스 종횡비와 Vignette 설정 불일치 (특히 9:16 세로 영상)
- **해결**:
  1. Vignette → Roundness: -100 ~ +100 조정
  2. 세로 영상(9:16)은 Roundness를 음수로
  3. 또는 Mask로 직접 그리기:
     - Effects → Video Effects → Generate → 4-Point Polygon Mask
     - 수동으로 영역 지정
- **확인**: 다양한 종횡비 (16:9, 9:16, 1:1)에서 테스트

**문제 7: 색보정 후 비디오 노이즈가 심해짐**
- **원인**: 어두운 영상의 Shadows/Exposure를 높여 숨어있던 노이즈 드러남
- **해결**:
  1. Effects → Noise Reduction & Grain → Denoise 적용
  2. Amount: 5-15 (적당히)
  3. 또는 Neat Video 플러그인 (강력, 유료)
  4. 촬영 단계 개선: 낮은 ISO, 충분한 조명
- **타협**: 약간의 노이즈는 영화적 질감으로 받아들임 (Film Grain)
- **주의**: 과도한 Denoise는 디테일 손실
