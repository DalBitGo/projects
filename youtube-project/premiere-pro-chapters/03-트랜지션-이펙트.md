### Chapter 3: 트랜지션 & 이펙트

#### 3.1 트랜지션(Transition) 이해하기

**트랜지션이란?**
- 두 클립 사이의 장면 전환 효과
- 자연스러운 흐름 연출
- 과도한 사용은 지양 (전문적이지 않아 보임)

**트랜지션의 종류**
- **Cross Dissolve**: 가장 많이 사용, 자연스러운 페이드
- **Dip to Black/White**: 시간/장소 변화 암시
- **Wipe**: 한쪽에서 다른 쪽으로 밀어내기
- **Zoom**: 확대/축소 효과
- **Slide/Push**: 슬라이드 효과

#### 3.2 트랜지션 적용하기

**기본 적용 방법**

**방법 1: 드래그 앤 드롭**
1. Effects 패널 열기 (Window → Effects)
2. "Video Transitions" 폴더 확장
3. Dissolve → Cross Dissolve 찾기
4. 클립 사이 또는 클립 끝에 드래그

**방법 2: 단축키 (가장 빠름!)**
- `Ctrl + D`: 기본 트랜지션(Cross Dissolve) 적용
- 클립 사이 또는 끝 부분 선택 후 실행

**트랜지션 적용 위치**
- **클립 사이**: 두 클립 사이에 페이드 효과
- **클립 시작**: 검은 화면에서 페이드 인
- **클립 끝**: 페이드 아웃 효과

**트랜지션 길이 조절**
- 트랜지션 양쪽 끝 드래그로 길이 조절
- 기본값: 1초 (30프레임)
- 권장: 0.5초 (자연스러운 길이)

#### 3.3 트랜지션 세부 설정

**Effect Controls에서 조정**
1. 트랜지션 클릭
2. Effect Controls 패널에서 설정 조정
3. 주요 옵션:
   - **Duration**: 트랜지션 지속 시간
   - **Alignment**: 시작, 중앙, 끝 정렬
   - **Border Width/Color**: 경계선 두께 및 색상 (일부 트랜지션)

**커스텀 설정**
- 각 트랜지션마다 고유 설정 존재
- 예: Wipe에서 방향 조절
- 실험적으로 조정하며 결과 확인

**기본 트랜지션 변경**
- Effects 패널에서 원하는 트랜지션 우클릭
- "Set Selected as Default Transition"
- 이후 Ctrl + D로 해당 트랜지션 적용

#### 3.4 비디오 이펙트(Video Effects) 기초

**이펙트란?**
- 클립의 시각적 속성 변경
- 색상, 블러, 왜곡, 스타일화 등
- 창의적 표현 및 문제 해결

**이펙트 카테고리**
- **Adjust**: 밝기, 대비, 채도 조정
- **Blur & Sharpen**: 블러, 선명도
- **Color Correction**: 색 보정 (Lumetri Color)
- **Distort**: 왜곡, 변형
- **Stylize**: 스타일 효과 (글로우, 모자이크 등)
- **Transform**: 위치, 크기, 회전
- **Video Effects**: 기타 효과

#### 3.5 이펙트 적용하기

**기본 적용 방법**
1. Effects 패널에서 원하는 이펙트 찾기
2. 클립에 드래그 앤 드롭
3. Effect Controls 패널에서 조정

**자주 사용하는 이펙트**

**1. Gaussian Blur (가우시안 블러)**
- 경로: Video Effects → Blur & Sharpen
- 용도: 배경 흐리게, 얼굴 모자이크
- 조정: Blurriness 값 증가 (10-50)

**2. Brightness & Contrast (밝기 & 대비)**
- 경로: Video Effects → Color Correction
- 용도: 어두운 영상 밝게, 선명도 증가
- 조정: Brightness +10~30, Contrast +10~20

**3. Crop (크롭)**
- 경로: Video Effects → Transform
- 용도: 영상 가장자리 자르기, 종횡비 조정
- 조정: Left, Right, Top, Bottom % 조절

**4. Transform (변형)**
- 경로: Video Effects → Distort
- 용도: 화면 내 위치/크기 조정 (PIP 효과)
- 조정: Position, Scale, Rotation

**5. Sharpen (선명도)**
- 경로: Video Effects → Blur & Sharpen
- 용도: 흐릿한 영상 선명하게
- 조정: Amount 10-50 (과도하면 부자연스러움)

#### 3.6 키프레임(Keyframe) 기초

**키프레임이란?**
- 시간에 따른 값 변화 기록
- 애니메이션의 핵심
- 시작점과 끝점을 설정하면 자동으로 중간 값 생성

**키프레임 설정 방법**

**기본 워크플로우**
1. Effect Controls 패널 열기
2. 속성 왼쪽의 **스톱워치 아이콘** 클릭 (키프레임 활성화)
3. 타임라인 헤드를 시작 위치로 이동
4. 값 설정 (예: Opacity 0%)
5. 타임라인 헤드를 끝 위치로 이동
6. 값 변경 (예: Opacity 100%)
7. 자동으로 키프레임 생성 및 애니메이션

**키프레임 실습: 페이드 인 효과**
1. 클립 선택
2. Effect Controls → Opacity의 스톱워치 클릭
3. 클립 시작 지점에서 Opacity 0%
4. 1초 후 위치에서 Opacity 100%
5. 재생하면 서서히 나타나는 효과

**키프레임 탐색**
- 화살표 아이콘: 이전/다음 키프레임으로 이동
- 다이아몬드 아이콘: 현재 위치에 키프레임 추가/삭제

#### 3.7 실전 이펙트: 자주 사용하는 효과 5가지

**1. 줌인/줌아웃 효과**
- **목적**: 역동적인 장면, 강조
- **방법**:
  1. 클립 선택 → Effect Controls
  2. Motion → Scale 스톱워치 클릭
  3. 시작: Scale 100%
  4. 끝: Scale 120% (줌인) 또는 80% (줌아웃)

**2. 화면 위치 이동 (Pan)**
- **목적**: 카메라 이동 효과
- **방법**:
  1. Motion → Position 스톱워치 클릭
  2. 시작: 960, 540 (중앙)
  3. 끝: 1200, 540 (오른쪽 이동)

**3. 페이드 인/아웃**
- **목적**: 부드러운 시작/종료
- **방법**:
  1. Opacity 스톱워치 클릭
  2. 페이드 인: 0% → 100%
  3. 페이드 아웃: 100% → 0%

**4. 슬로우 모션 (속도 조절)**
- **목적**: 강조, 감성적 장면
- **방법**:
  1. 클립 우클릭 → Speed/Duration
  2. Speed: 50% (절반 속도)
  3. "Ripple Edit" 체크 시 뒤 클립 자동 조정

**5. 역재생**
- **목적**: 창의적 효과
- **방법**:
  1. 클립 우클릭 → Speed/Duration
  2. "Reverse Speed" 체크

#### 3.8 이펙트 관리

**이펙트 복사/붙여넣기**
- 클립 선택 → Ctrl + C
- 다른 클립 선택 → Ctrl + V
- 또는: 이펙트만 복사 (Effect Controls에서 이펙트 우클릭 → Copy)

**이펙트 켜기/끄기**
- Effect Controls에서 fx 아이콘 클릭
- 적용 전후 비교 용이

**이펙트 삭제**
- Effect Controls에서 이펙트 선택 → Delete
- 또는 우클릭 → Clear

**이펙트 프리셋 저장**
- Effect Controls에서 설정 완료 후
- 이펙트 이름 우클릭 → Save Preset
- 재사용 가능

#### 3.9 실습 프로젝트 2: 트랜지션과 이펙트로 다이나믹한 영상 만들기

**목표**: Chapter 2에서 만든 영상에 트랜지션과 이펙트 추가

**단계별 실습**
1. **트랜지션 추가**
   - 클립 사이에 Cross Dissolve (Ctrl + D)
   - 시작 부분 페이드 인, 끝 부분 페이드 아웃

2. **줌인 효과 추가**
   - 중요한 클립에 Scale 키프레임 (100% → 110%)

3. **색감 보정**
   - 어두운 클립에 Brightness & Contrast 적용

4. **속도 조절**
   - 한 클립을 슬로우 모션 (50%)으로 변경

**체크리스트**
- [ ] 클립 사이 Cross Dissolve 추가
- [ ] 시작/끝 페이드 효과
- [ ] 최소 1개 클립에 줌인 효과
- [ ] 색감 보정 적용
- [ ] 프로젝트 저장

**팁**
- 트랜지션은 모든 클립 사이에 넣지 말 것 (2-3곳만)
- 이펙트는 과하지 않게 (자연스러움이 우선)
- 재생하며 효과 확인 필수

#### 3.10 자주 하는 실수와 해결법

**실수 1: 트랜지션 적용 시 "부족한 미디어" 경고**
- **원인**: 클립 끝부분에 여유 프레임(핸들) 부족
- **증상**: 빨간 경고 또는 트랜지션이 제대로 작동 안 함
- **해결**:
  - 클립을 약간 길게 남겨두기 (양쪽 0.5초씩)
  - 또는 트랜지션 길이 줄이기 (0.3초 이하)
  - 또는 Alignment를 "End at Cut" 또는 "Start at Cut"으로 변경
- **예방**: 소스 클립을 In/Out 포인트보다 여유있게 자르기

**실수 2: 모든 클립 사이에 트랜지션 넣어서 촌스러움**
- **원인**: 트랜지션 과다 사용
- **해결**:
  - 의미있는 장면 전환에만 사용 (시간/장소 변화)
  - 대부분은 하드 컷(바로 전환)이 더 전문적
  - 한 영상에 2-3개 정도만 사용
- **원칙**: "Less is More" - 절제가 프로페셔널함

**실수 3: 키프레임이 너무 많아서 움직임이 부자연스러움**
- **원인**: 너무 많은 키프레임 또는 Ease 미적용
- **해결**:
  - 필요한 키프레임만 남기고 삭제
  - 키프레임 우클릭 → Ease In/Ease Out 적용
  - 또는 Bezier 곡선으로 부드럽게 조정
- **팁**: 2-3개 키프레임으로도 충분한 효과 가능

**실수 4: 이펙트를 적용했는데 효과가 안 보임**
- **원인**: 이펙트 값이 너무 작거나 잘못된 클립에 적용
- **해결**:
  - Effect Controls에서 값 확대 (예: Blur 10 → 50)
  - 올바른 클립이 선택되었는지 확인
  - fx 아이콘이 꺼져있지 않은지 확인
- **디버깅**: Effect Controls 패널에서 이펙트 목록 확인

**실수 5: 슬로우 모션 적용 후 영상이 끊김**
- **원인**: 원본 프레임 레이트 낮음 또는 보간 설정 문제
- **해결**:
  - Speed/Duration → Time Interpolation: Optical Flow 선택
  - 또는 원본을 60fps 이상으로 촬영
  - Frame Blending 활성화
- **한계**: 30fps 영상을 10%로 늦추면 끊김 불가피

**실수 6: 이펙트 여러 개 적용 시 렌더링이 느림**
- **원인**: GPU 부하, 복잡한 이펙트 중첩
- **해결**:
  - 편집 중 일부 이펙트 비활성화 (fx 아이콘)
  - 또는 Render In to Out (Enter 키)
  - 프록시 파일 사용
- **최종**: 내보내기 전에만 모든 이펙트 활성화

**실수 7: 트랜지션 양쪽 클립의 밝기가 달라 어색함**
- **원인**: 색감/밝기 불일치
- **해결**:
  - 두 클립에 동일한 Lumetri Color 프리셋 적용
  - 또는 Adjustment Layer로 일관성 부여
  - Dip to White/Black 트랜지션 사용 (밝기 차이 완화)
- **원칙**: 트랜지션 전후 클립 톤 맞추기

#### 3.11 프로 팁 (이펙트 & 트랜지션 고급)

**팁 1: 기본 트랜지션 커스터마이징**
- Cross Dissolve에 Directional Blur 추가
- 방향성 있는 디졸브 효과 연출
- **방법**:
  1. Cross Dissolve 적용
  2. 트랜지션 클릭 → Effect Controls
  3. Custom 설정에서 Direction 조정

**팁 2: 이즈 커브(Ease Curve) 완전 제어**
- 키프레임 우클릭 → Temporal Interpolation → Bezier
- 핸들 드래그하여 가속도 곡선 조정
- **효과**: 영화 같은 부드러운 움직임
- **예**: 빠르게 시작 → 천천히 멈춤 (자연스러운 모션)

**팁 3: 여러 속성 동시 애니메이션**
- Position + Scale + Rotation 동시 키프레임
- 복합적인 모션 그래픽 효과
- **예시**: 텍스트가 회전하며 확대되며 이동
- **주의**: 너무 복잡하면 산만함

**팁 4: 이펙트 프리셋 라이브러리 구축**
- 자주 쓰는 설정을 프리셋으로 저장
- 폴더 구조:
  ```
  Effects Presets/
  ├── 컬러_따뜻한톤
  ├── 컬러_차가운톤
  ├── 블러_배경용
  ├── 줌인_강조용
  └── 글로우_네온
  ```
- 원클릭으로 일관된 스타일 적용

**팁 5: 트랜지션 속도 커브 조정**
- 일부 트랜지션은 속도 커브 지원
- Effect Controls → Custom → Ease In/Out 슬라이더
- 선형보다 곡선 전환이 자연스러움
- **실험**: 다양한 커브로 느낌 테스트

**팁 6: Morph Cut으로 Jump Cut 부드럽게**
- 같은 장면 점프 컷 사이에 적용
- Effects → Video Transitions → Morph Cut
- 자연스러운 얼굴/몸 전환
- **용도**: 인터뷰, Vlog에서 편집점 숨기기
- **제한**: 배경 변화 크면 효과 약함

**팁 7: 이펙트 스택 순서 최적화**
- Effect Controls에서 이펙트 순서 드래그로 변경
- **권장 순서**:
  1. 크롭/변형 (Transform, Crop)
  2. 색 보정 (Lumetri Color)
  3. 블러/선명도 (Blur, Sharpen)
  4. 스타일 효과 (Glow, Vignette)
- **이유**: 순서에 따라 최종 결과 달라짐

#### 3.12 고급 키프레임 기법

**속도 그래프 vs 값 그래프**
- Effect Controls → 그래프 아이콘 클릭
- **속도 그래프**: 변화 속도 제어 (가속/감속)
- **값 그래프**: 실제 값 변화 제어
- **활용**: 속도 그래프로 더 자연스러운 모션

**키프레임 보간 모드**
- **Linear**: 일정한 속도 (딱딱함)
- **Bezier**: 곡선 (부드러움, 가장 많이 사용)
- **Hold**: 값 유지 후 갑자기 변화 (스톱 모션 느낌)
- **Auto Bezier**: 자동 부드러움
- **우클릭 → Temporal Interpolation에서 선택**

**키프레임 복사 및 반전**
- 키프레임 선택 → Ctrl + C
- 다른 시간에 Ctrl + V
- 또는 우클릭 → Copy → Paste Reverse (반대 효과)
- **예**: 줌인 키프레임 복사 → 반전 → 줌아웃

**키프레임 일괄 이동/스케일**
- 여러 키프레임 선택 (Shift + 클릭)
- 드래그하여 전체 타이밍 조정
- Alt + 드래그: 간격 유지하며 이동
- **효과**: 애니메이션 템포 조절

#### 3.13 실전 예시: 인트로 타이틀 애니메이션

**시나리오**: 5초 인트로 with 텍스트 + 로고 애니메이션

**애니메이션 설계**

**0-1초: 로고 줌인**
- Scale: 50% → 100%
- Opacity: 0% → 100%
- Ease Out 적용

**1-3초: 로고 유지**
- 키프레임 없음 (정지)

**3-4초: 텍스트 슬라이드 인**
- Position: X = -500 → X = 960
- Ease In 적용

**4-5초: 전체 페이드 아웃**
- Opacity: 100% → 0%
- Ease In 적용

**추가 효과**
- 로고에 Drop Shadow (깊이감)
- 텍스트에 Glow (빛나는 효과)
- 배경에 Gradient Wipe 트랜지션

**제작 시간**: 10분

#### 3.14 트랜지션 종류별 사용 가이드

**Cross Dissolve (크로스 디졸브)**
- **사용 시기**: 시간 경과, 부드러운 전환
- **길이**: 0.5-1초
- **피해야 할 때**: 액션 장면 (끊어짐)
- **추천**: 80% 경우에 이것만 사용

**Dip to Black/White**
- **사용 시기**: 챕터 전환, 큰 시간/장소 변화
- **길이**: 1-2초
- **효과**: 극적인 분위기
- **예**: 하루가 바뀌거나 회상 장면

**Wipe (와이프)**
- **사용 시기**: 멀티캠 전환, 창의적 효과
- **길이**: 0.5초
- **주의**: 과도하면 구식 느낌
- **현대적 사용**: 빠른 속도 (0.3초)

**Zoom (줌)**
- **사용 시기**: 강조, 역동적 전환
- **길이**: 0.3-0.5초
- **조합**: 음악 비트와 동기화
- **예**: 뮤직 비디오, 하이라이트

**Push/Slide**
- **사용 시기**: 비교 영상, 뉴스 스타일
- **길이**: 0.5초
- **현대성**: 최근 덜 사용됨
- **대안**: 하드 컷이 더 세련됨

**Film Dissolve**
- **사용 시기**: 빈티지, 레트로 느낌
- **효과**: 필름 질감 추가
- **조합**: 컬러 그레이딩 (세피아, 흑백)

#### 3.15 문제 해결 (이펙트 관련)

**문제 1: 키프레임 애니메이션이 재생 시 끊김**
- **원인**: 렌더링 부족, 프리뷰 해상도 낮음
- **해결**:
  - Sequence → Render In to Out (Enter)
  - 프리뷰 해상도 1/2 또는 Full로 변경
  - GPU 가속 확인
- **임시**: 편집 중엔 1/4 해상도, 최종 확인 시 Full

**문제 2: 이펙트 값을 바꿨는데 변화가 없음**
- **원인**: 키프레임이 활성화되어 있어 자동 조정 안 됨
- **해결**:
  - 스톱워치 아이콘 다시 클릭 (키프레임 비활성화)
  - 또는 모든 키프레임 삭제
  - 값 조정 후 재생 확인
- **이해**: 키프레임 활성 시 타임라인 위치마다 값 다름

**문제 3: 트랜지션이 한쪽 클립에만 적용됨**
- **원인**: 클립 끝이 아닌 중간에 적용
- **해결**:
  - 정확히 클립 경계에 드롭
  - 또는 트랜지션 Alignment 조정
- **확인**: Effect Controls에서 Alignment 설정

**문제 4: 복사한 이펙트가 제대로 붙여넣기 안 됨**
- **원인**: 클립 유형 불일치 (비디오/오디오)
- **해결**:
  - 동일 유형 클립에만 붙여넣기
  - 또는 개별 이펙트만 복사 (Effect Controls에서)
- **대안**: 프리셋 저장 후 적용

**문제 5: Morph Cut이 이상하게 작동함 (왜곡)**
- **원인**: 배경 변화, 움직임 과다
- **해결**:
  - 배경 동일한 컷에만 사용
  - 짧은 길이 (0.5초 이하)
  - 또는 Cross Dissolve로 대체
- **최적**: 고정 카메라 인터뷰

#### 3.16 연습 과제 (트랜지션 & 이펙트 마스터)

**기초 과제 1: 페이드 인/아웃 완벽 구현**
- 목표: Opacity 키프레임 숙달
- 요구사항:
  - 영상 시작 페이드 인 (0 → 100%, 1초)
  - 영상 끝 페이드 아웃 (100% → 0%, 1초)
  - Ease In/Out 적용
- 시간 제한: 5분

**중급 과제 2: 5가지 트랜지션 비교 영상**
- 준비물: 동일한 2개 클립
- 목표: 트랜지션별 특성 이해
- 단계:
  1. 같은 클립 5번 복제
  2. 각 사이에 다른 트랜지션 적용
     - Cross Dissolve
     - Dip to Black
     - Wipe
     - Zoom
     - Push
  3. 각 트랜지션 길이 0.5초 통일
- 비교 분석: 어떤 상황에 어울리는지 메모
- 시간 제한: 15분

**고급 과제 3: 모션 그래픽 제목 제작**
- 목표: 복합 키프레임 애니메이션
- 요구사항:
  - 텍스트 레이어 생성
  - 3가지 속성 동시 애니메이션:
    1. Position (좌 → 중앙)
    2. Scale (50% → 100%)
    3. Rotation (45° → 0°)
  - 모든 키프레임에 Ease 적용
  - 총 애니메이션 길이 2초
- 시간 제한: 20분

**창의성 과제: 시그니처 트랜지션 개발**
- 목표: 독창적인 전환 효과 제작
- 자유 기법:
  - 여러 트랜지션 중첩
  - 이펙트 + 트랜지션 조합
  - 키프레임 애니메이션 활용
- **예시 아이디어**:
  - 회전하며 줌아웃 + Cross Dissolve
  - 블러 증가 → 전환 → 블러 감소
  - RGB 분리 효과 + Wipe
- 재사용 가능하도록 프리셋 저장
- **보너스**: 3개 영상에 동일 트랜지션 적용하여 일관성 확인
