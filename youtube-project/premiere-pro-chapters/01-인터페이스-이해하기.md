### Chapter 1: 인터페이스 이해하기

#### 1.1 Premiere Pro 시작하기
**프로그램 실행 및 시작 화면**
- Adobe Creative Cloud에서 Premiere Pro 실행
- 시작 화면에서 "새 프로젝트" 또는 "프로젝트 열기" 선택
- 최근 프로젝트 목록 확인 및 빠른 접근

**새 프로젝트 생성**
- 프로젝트 이름 설정 (예: "내 첫 영상 편집")
- 저장 위치 선택 (작업 폴더 구조 권장: `내 문서/Premiere Projects/프로젝트명`)
- 프로젝트 설정:
  - 비디오 렌더링 및 재생 → 렌더러: Mercury Playback Engine GPU 가속
  - 오디오 → 샘플 레이트: 48000 Hz (기본값)

#### 1.2 작업 공간(Workspace) 구성 이해하기

**기본 패널 구성**
1. **프로젝트 패널 (Project Panel)** - 좌측 하단
   - 임포트한 미디어 파일 관리
   - 폴더 구조로 정리 (예: 영상소스, 오디오, 이미지)
   - 더블클릭으로 소스 모니터에서 미리보기

2. **소스 모니터 (Source Monitor)** - 좌측 상단
   - 개별 클립 미리보기 및 편집
   - In/Out 포인트 설정으로 사용할 구간 지정
   - 단축키: I (In 포인트), O (Out 포인트)

3. **프로그램 모니터 (Program Monitor)** - 우측 상단
   - 최종 편집 결과물 실시간 미리보기
   - 타임라인의 재생 헤드 위치 영상 표시
   - 안전 영역, 그리드 등 오버레이 표시 가능

4. **타임라인 패널 (Timeline Panel)** - 하단
   - 실제 편집 작업이 이루어지는 공간
   - 비디오/오디오 트랙으로 구성
   - 클립 배치, 자르기, 효과 적용

5. **이펙트 컨트롤 (Effect Controls)** - 좌측 상단 탭
   - 선택한 클립의 속성 조정
   - 위치, 스케일, 회전, 불투명도 등
   - 키프레임을 통한 애니메이션 제어

6. **이펙트 패널 (Effects Panel)** - 좌측 하단
   - 비디오/오디오 이펙트 및 트랜지션 라이브러리
   - 검색 기능으로 빠른 찾기
   - 드래그 앤 드롭으로 적용

**작업 공간 프리셋**
- **편집 (Editing)**: 기본 편집 작업에 최적화
- **컬러 (Color)**: Lumetri Color 패널 중심 배치
- **이펙트 (Effects)**: 이펙트 작업에 집중
- **오디오 (Audio)**: 오디오 편집 및 믹싱
- **그래픽 (Graphics)**: 타이틀 및 모션 그래픽

**작업 공간 커스터마이징**
- Window 메뉴에서 원하는 패널 추가/제거
- 패널 탭을 드래그하여 위치 변경
- 패널 경계선 드래그로 크기 조절
- 커스텀 작업 공간 저장: Window → Workspaces → Save as New Workspace

#### 1.3 시퀀스(Sequence) 이해하기

**시퀀스란?**
- 타임라인에서 편집 작업을 담는 컨테이너
- 하나의 프로젝트에 여러 시퀀스 생성 가능
- 최종 영상의 해상도, 프레임 레이트 등 결정

**새 시퀀스 생성 방법**

**방법 1: 자동 생성 (권장 - 초보자)**
- 첫 번째 클립을 타임라인에 드래그
- 클립 설정에 맞춰 자동으로 시퀀스 생성
- 가장 간단하고 실수가 적음

**방법 2: 수동 생성**
- File → New → Sequence
- 프리셋 선택:
  - **YouTube Shorts/TikTok**: ARRI → 1080p, 9:16 세로 포맷
  - **YouTube 일반**: Digital SLR → 1080p, 1920x1080
  - **고품질**: 4K UHD 3840x2160

**주요 시퀀스 설정**
- **해상도**: 1920x1080 (FHD), 3840x2160 (4K)
- **프레임 레이트**:
  - 24fps: 영화 느낌
  - 30fps: 일반 영상
  - 60fps: 스포츠, 게임, 부드러운 영상
- **픽셀 종횡비**: Square Pixels (1.0)

#### 1.4 필수 단축키 (Chapter 1 단계)

**재생 및 탐색**
- `스페이스바`: 재생/정지
- `L`: 재생 속도 증가 (최대 3배속)
- `J`: 역재생
- `K`: 정지
- `→/←`: 프레임 단위 이동
- `Shift + →/←`: 5프레임 단위 이동

**작업 공간**
- `~`: 마우스 커서 아래 패널 전체 화면 전환
- `` ` ``: 작업 공간 전환 (순환)

**기타**
- `Ctrl + S`: 프로젝트 저장
- `Ctrl + Z`: 실행 취소
- `Ctrl + Shift + Z`: 다시 실행

#### 1.5 프로젝트 관리 기본

**폴더 구조 권장 사항**
```
프로젝트명/
├── Premiere/          # .prproj 파일
├── Raw Footage/       # 원본 영상
├── Audio/            # 오디오 파일
│   ├── BGM/         # 배경음악
│   ├── SFX/         # 효과음
│   └── VO/          # 보이스오버
├── Graphics/         # 이미지, 그래픽
│   ├── Logos/       # 로고
│   ├── Thumbnails/  # 썸네일
│   └── Assets/      # 기타 소스
├── Exports/          # 내보낸 영상
│   ├── Finals/      # 최종본
│   └── Drafts/      # 초안
└── Project Files/    # 기타 파일
    ├── Scripts/     # 대본
    └── References/  # 참고 자료
```

**프로젝트 명명 규칙**
- 날짜_프로젝트명 형식 권장: `2025-01-15_MyFirstVlog`
- 버전 관리: `MyProject_v01`, `MyProject_v02`, `MyProject_Final`
- 공백 대신 언더스코어(_) 또는 하이픈(-) 사용

**자동 저장 설정**
- Edit → Preferences → Auto Save
- 자동 저장 간격: 5-10분 권장
- 최대 버전 수: 10-20개
- 자동 저장 위치 확인: Documents/Adobe/Premiere Pro/[버전]/

**자동 저장 복구 방법**
1. Premiere Pro 크래시 후 재실행
2. "프로젝트 복구" 대화상자 확인
3. 가장 최근 자동 저장 파일 선택
4. 다른 이름으로 저장 권장

#### 1.6 인터페이스 최적화 팁

**성능 향상 설정**
1. **미디어 캐시 위치 변경**
   - Edit → Preferences → Media Cache
   - SSD 드라이브에 캐시 저장 (HDD보다 빠름)
   - 정기적으로 캐시 정리

2. **GPU 가속 활성화**
   - Edit → Preferences → Playback
   - Renderer: Mercury Playback Engine GPU Acceleration
   - CUDA (NVIDIA) 또는 OpenCL (AMD) 선택

3. **메모리 할당**
   - Edit → Preferences → Memory
   - Other Applications에 2-4GB 정도 남기고 나머지 Premiere에 할당

**듀얼 모니터 설정**
- 모니터 1: 소스/프로그램 모니터 (대형)
- 모니터 2: 타임라인, 프로젝트 패널, 이펙트
- 패널을 드래그하여 다른 모니터로 이동 가능

#### 1.7 자주 하는 실수와 해결법

**실수 1: 클립을 찾을 수 없음 (Media Offline)**
- **원인**: 원본 파일 이동 또는 삭제
- **해결**: 우클릭 → Link Media → 파일 위치 재지정
- **예방**: 프로젝트 폴더에 모든 소스 파일 복사

**실수 2: 재생이 끊김**
- **원인**: 고해상도 파일, 부족한 컴퓨터 성능
- **해결**:
  - 프록시 생성 (저해상도 복사본)
  - 타임라인 해상도를 1/4 또는 1/2로 낮춤 (프로그램 모니터 → 재생 해상도)
  - Sequence → Render In to Out (Enter)

**실수 3: 시퀀스 설정이 클립과 다름**
- **증상**: 클립에 빨간 선 표시
- **해결**: 시퀀스 우클릭 → Sequence Settings → 클립과 동일하게 설정
- **또는**: 클립을 먼저 드래그하여 자동 시퀀스 생성

**실수 4: 저장을 안 함**
- **예방**: Ctrl + S 습관화
- **자동 저장 활성화 필수**
- 중요한 작업 전 수동 저장

#### 1.8 패널 상세 가이드

**프로젝트 패널 고급 활용**
- **리스트 뷰 vs 아이콘 뷰**: 하단 아이콘으로 전환
- **정렬**: 열 제목 클릭 (이름, 날짜, 길이 등)
- **검색**: 검색 바에 키워드 입력
- **라벨 색상**: 클립 우클릭 → Label → 색상 지정 (정리용)
- **메타데이터**: 클립 선택 → 메타데이터 패널에서 정보 확인/편집

**타임라인 패널 확대/축소**
- `+/-`: 시간축 확대/축소
- `\`: 전체 시퀀스 보기 (Fit)
- 마우스 휠: 위아래 스크롤
- Alt + 마우스 휠: 확대/축소 (프레임 단위 편집 시 유용)

**프로그램 모니터 오버레이**
- 버튼 클릭하여 오버레이 표시:
  - **Safe Margins**: 안전 영역 (TV 방송용)
  - **Grid**: 격자 (구도 확인)
  - **Rulers**: 눈금자
