### Chapter 5: 오디오 편집

#### 5.1 오디오 기초 이해

**오디오의 중요성**
- "사람들은 나쁜 영상은 봐도, 나쁜 오디오는 못 본다"
- 음질이 영상 품질의 50% 이상 차지
- 배경 소음, 음량 불균형은 시청자 이탈 원인

**오디오 트랙 구성**
- **A1, A2**: 주 오디오 (대사, 내레이션)
- **A3, A4**: BGM (배경음악)
- **A5, A6**: 효과음 (SFX)

**오디오 미터 (Audio Meter) 확인**
- Window → Audio Meters
- dB 단위로 음량 표시
- **녹색**: 안전 영역
- **노란색**: 주의 (-6dB ~ -3dB)
- **빨간색**: 클리핑(왜곡) 발생 (-0dB 초과)

#### 5.2 오디오 레벨 조정

**목표 음량 가이드**
- **대사/내레이션**: -12dB ~ -6dB
- **BGM**: -20dB ~ -18dB (대사 방해 안 되도록)
- **효과음**: -15dB ~ -10dB
- **최종 마스터**: -3dB ~ -1dB (피크)

**방법 1: 타임라인에서 직접 조정**
1. 클립 선택
2. 오디오 파형 위쪽 선 (볼륨 라인) 드래그
3. 위로: 음량 증가, 아래로: 음량 감소
4. 숫자 표시: dB 변화량

**방법 2: Effect Controls 패널**
1. 클립 선택
2. Effect Controls → Volume → Level
3. 슬라이더 또는 숫자 입력으로 조정
4. 정밀 조정 가능

**방법 3: Audio Clip Mixer (실시간)**
- Window → Audio Clip Mixer
- 여러 트랙 동시 조정
- 라이브 믹싱 느낌

**키프레임으로 음량 변화**
1. Effect Controls → Volume → Level 스톱워치 클릭
2. 시작 지점: -10dB
3. 끝 지점: -20dB (페이드 아웃)
4. 자동으로 부드러운 전환

#### 5.3 배경음악(BGM) 추가하기

**BGM 소스**
- **무료**: YouTube Audio Library, Pixabay, Free Music Archive
- **유료**: Epidemic Sound, Artlist, Soundstripe
- **주의**: 저작권 확인 필수!

**BGM 삽입 워크플로우**
1. BGM 파일 임포트
2. A3 트랙에 배치 (대사와 분리)
3. 영상 길이에 맞춰 자르기
4. 음량 조정: -20dB ~ -18dB

**BGM 페이드 인/아웃**
1. 클립 시작/끝 부분 선택
2. Effect Controls → Volume → Level
3. 키프레임 설정:
   - 페이드 인: -∞dB → -20dB (2-3초)
   - 페이드 아웃: -20dB → -∞dB (2-3초)

**빠른 페이드 (Constant Power)**
- BGM 클립 끝 부분 우클릭
- Apply Default Audio Transition (Ctrl + Shift + D)
- 자동으로 Constant Power 트랜지션 적용
- 길이 조절 가능

#### 5.4 오디오 트랜지션

**Constant Power (가장 많이 사용)**
- 부드러운 페이드 인/아웃
- 클립 사이 또는 끝에 적용
- 단축키: Ctrl + Shift + D

**Constant Gain**
- 선형적 페이드
- 덜 부드러움, 특별한 경우에만 사용

**Crossfade**
- 두 오디오 클립 사이 자연스러운 전환
- 음악 교체 시 유용

**적용 방법**
1. Effects → Audio Transitions → Crossfade
2. 원하는 트랜지션을 클립에 드래그
3. 길이 조절 (0.5-2초 권장)

#### 5.5 효과음(SFX) 삽입

**효과음 활용**
- 트랜지션 강조 (whoosh 소리)
- 텍스트 등장/사라질 때
- 버튼 클릭, 알림 등 UI 효과

**효과음 소스**
- **무료**: Freesound.org, Zapsplat, BBC Sound Effects
- **유료**: Epidemic Sound, Motion Array

**삽입 방법**
1. 효과음 파일 임포트
2. A5 트랙에 배치
3. 정확한 타이밍에 맞춰 위치 조정
4. 음량: -15dB ~ -10dB
5. 길이 조절 (대부분 1초 이내)

**팁**
- 과도하게 사용하지 말 것
- 중요한 장면에만 사용
- 음량이 너무 크면 거슬림

#### 5.6 오디오 이펙트

**Essential Sound 패널 (권장)**
- Window → Essential Sound
- 초보자도 쉽게 사용 가능한 프리셋

**클립 타입 지정**
1. 오디오 클립 선택
2. Essential Sound → 타입 선택:
   - **Dialogue**: 대화/내레이션
   - **Music**: 배경음악
   - **SFX**: 효과음
   - **Ambience**: 환경음

**Dialogue (대화) 편집**
- **Loudness**: 자동 음량 정규화
  - Target Loudness: -16 LUFS (YouTube 권장)
- **Repair**: 노이즈 제거
  - Reduce Noise: 배경 소음 제거 (20-30%)
  - Reduce Rumble: 저음 럼블 제거
  - DeHum: 전기 험(hum) 제거
- **Clarity**: 음성 선명도
  - Dynamics: 다이나믹 레인지 압축
  - EQ: 음성 주파수 강조

**Music (음악) 편집**
- **Ducking**: 자동 볼륨 감소
  - 대화 트랙 지정 시 자동으로 BGM 음량 감소
  - 대화 없을 때 자동으로 복구
  - Duck Amount: 15-25dB 권장

#### 5.7 오디오 덕킹 (Auto-Ducking) 설정

**덕킹이란?**
- 대화가 있을 때 BGM 자동으로 낮춤
- 대화 끝나면 BGM 다시 복구
- 수동 키프레임 작업 불필요

**설정 방법**
1. BGM 클립 선택
2. Essential Sound → Music 선택
3. Ducking 섹션 펼치기
4. "Duck Against" → 대화 트랙 선택 (A1)
5. 설정 조정:
   - **Duck Amount**: 20dB (BGM이 얼마나 줄어드는지)
   - **Fade**: 0.5초 (전환 속도)
   - **Sensitivity**: Medium

**수동 덕킹 (키프레임 방식)**
1. BGM 클립 선택
2. Effect Controls → Volume → Level 스톱워치
3. 대사 시작 지점: -25dB (낮춤)
4. 대사 끝 지점: -18dB (복구)
5. 반복

#### 5.8 노이즈 제거

**배경 소음 문제**
- 바람 소리, 에어컨, 냉장고 소음 등
- 촬영 시 방지가 최선이지만, 후처리 가능

**DeNoise (노이즈 제거)**
1. 대화 클립 선택
2. Essential Sound → Dialogue
3. Repair → Reduce Noise
4. Amount: 20-40% (너무 높으면 음성 왜곡)

**고급: Audition 연동**
- Premiere Pro보다 강력한 오디오 편집
- 클립 우클릭 → Edit Clip in Adobe Audition
- Audition에서 노이즈 제거 후 자동 반영

#### 5.9 오디오 동기화 (Audio Sync)

**시나리오**: 카메라 오디오 vs 외장 마이크

**자동 동기화**
1. 비디오 클립과 오디오 클립 동시 선택
2. 우클릭 → Synchronize
3. Audio 옵션 선택
4. OK → 자동으로 파형 일치시켜 동기화
5. 카메라 오디오 삭제 또는 음소거

**수동 동기화**
1. 파형 비교하며 위치 조정
2. 클랩(clap) 소리나 특정 소리 찾기
3. 정확히 맞춘 후 링크

#### 5.10 오디오 채널 및 스테레오

**모노 vs 스테레오**
- **모노**: 단일 채널 (Left = Right)
- **스테레오**: 좌우 채널 분리

**오디오 채널 확인**
- 클립 우클릭 → Modify → Audio Channels
- Mono 또는 Stereo 설정

**패닝 (Panning)**
- 소리를 좌우로 이동
- Audio Clip Mixer에서 Pan 슬라이더 조정
- 용도: 공간감 연출, 특수 효과

#### 5.11 실전 오디오 믹싱 워크플로우

**단계별 작업**

**1. 음량 균형 (5분)**
- 모든 대사 클립 음량 확인
- Audio Meters 보며 -12dB ~ -6dB 맞추기
- 너무 작거나 큰 부분 조정

**2. BGM 추가 및 조정 (3분)**
- BGM 임포트 후 A3 트랙 배치
- 음량 -20dB 설정
- 시작/끝 페이드 적용

**3. 덕킹 설정 (2분)**
- BGM → Essential Sound → Music
- Ducking 활성화, Duck Against A1
- Duck Amount 20dB

**4. 노이즈 제거 (3분)**
- 대사 클립 → Essential Sound → Dialogue
- Reduce Noise 30%
- Reduce Rumble 체크

**5. 효과음 추가 (2분)**
- 필요한 부분에 효과음 배치
- 음량 -12dB
- 타이밍 정밀 조정

**6. 최종 확인 (2분)**
- 전체 재생하며 청취
- Audio Meters로 피크 확인 (빨간색 없어야 함)
- 필요 시 마스터 볼륨 조정

**총 소요 시간: 약 15-20분**

#### 5.12 오디오 내보내기 팁

**YouTube 권장 설정**
- **포맷**: AAC
- **샘플 레이트**: 48 kHz
- **비트레이트**: 320 kbps (고음질)

**Loudness 정규화**
- YouTube 자동 정규화: -14 LUFS
- Essential Sound → Loudness → Auto-Match
- Target: -16 LUFS (YouTube 기준보다 약간 높게)

**목표**: 기존 영상에 BGM과 오디오 편집 적용

#### 5.14 자주 하는 실수와 해결법

**실수 1: BGM이 대사를 완전히 덮음**
- **원인**: BGM 음량이 너무 크거나 덕킹 미설정
- **해결**:
  - BGM 음량을 -20dB ~ -18dB로 낮춤
  - Essential Sound에서 Auto-Ducking 활성화
  - Duck Against를 대사 트랙(A1)으로 설정
  - Duck Amount를 20dB로 조정
- **예방**: BGM 추가 즉시 음량 확인 및 덕킹 설정 습관화
- **프로 기준**: 대사가 있을 때 BGM은 대사보다 최소 12dB 낮아야 함

**실수 2: 오디오 클리핑 (찌그러진 소리)**
- **원인**: 음량이 0dB를 초과하여 왜곡 발생
- **증상**: Audio Meter가 빨간색 표시, 소리가 깨짐
- **해결**:
  - Audio Meter에서 빨간색 구간 찾기
  - 해당 클립 선택 → Volume Level을 -3dB ~ -6dB 낮춤
  - 또는 Limiter 이펙트 적용 (Audio Effects → Amplitude and Compression → Limiter)
  - Max Threshold: -0.5dB 설정
- **예방**: 편집 중 Audio Meter를 항상 모니터링, 피크가 -3dB 이상 올라가지 않도록 관리

**실수 3: 페이드 인/아웃이 너무 급격함**
- **원인**: 트랜지션 길이가 너무 짧음 (0.5초 이하)
- **해결**:
  - Constant Power 트랜지션 길이를 2-3초로 연장
  - 또는 키프레임으로 수동 페이드 설정
  - Effect Controls → Volume → Level에서 부드러운 곡선 생성
- **권장 길이**:
  - 페이드 인: 2-3초 (자연스러운 시작)
  - 페이드 아웃: 3-4초 (여운 있는 마무리)
  - 빠른 전환: 1초 (다이나믹한 느낌)

**실수 4: 노이즈 제거 과다 (음성이 로봇 같음)**
- **원인**: Reduce Noise Amount를 50% 이상 높게 설정
- **해결**:
  - Reduce Noise를 20-30%로 낮춤
  - DeReverb와 함께 사용하지 말 것 (이중 처리 방지)
  - 원본과 비교하며 조금씩 조정
- **팁**: 노이즈 제거는 촬영 시 방지가 최선 (좋은 마이크 사용)
- **적정 수치**:
  - 가벼운 소음: 15-20%
  - 중간 소음: 25-35%
  - 심한 소음: Adobe Audition 활용 권장

**실수 5: 오디오와 비디오 동기화 어긋남**
- **원인**: 수동 조정 중 위치 오류, 드리프트 현상
- **증상**: 입 모양과 소리가 맞지 않음 (립싱크 문제)
- **해결**:
  - 클립 선택 → 우클릭 → Synchronize → Audio 옵션
  - 자동 파형 매칭으로 정확히 동기화
  - 또는 클랩(clap) 소리를 기준점으로 수동 정렬
- **예방**: 촬영 시 클랩보드 사용 또는 손뼉 치기
- **드리프트 방지**: 카메라와 외장 레코더의 샘플 레이트 일치 (48kHz)

**실수 6: 스테레오/모노 설정 오류**
- **증상**: 소리가 한쪽에서만 들림 또는 중복됨
- **원인**: 모노 오디오를 스테레오 트랙에 배치 시 채널 매핑 오류
- **해결**:
  - 클립 우클릭 → Modify → Audio Channels
  - Mono로 설정된 소스는 "Mono" 선택
  - 또는 Audio Channel Mapping에서 Left/Right 둘 다 채널 1로 설정
- **확인 방법**: 헤드폰으로 좌우 채널 각각 테스트

**실수 7: 효과음 타이밍이 영상과 맞지 않음**
- **원인**: 프레임 단위 정확도 부족
- **해결**:
  - 타임라인을 최대로 확대 (+키 여러 번)
  - 재생 헤드를 정확한 프레임에 위치
  - 효과음 클립을 프레임 단위로 이동 (←/→ 키)
  - Snap 기능 (S) 활용
- **팁**: 효과음 앞부분에 무음 구간이 있는지 확인 (있으면 자르기)
- **정밀 조정**: Alt + ←/→로 1프레임씩 이동

#### 5.15 프로 팁

**팁 1: 오디오 트랙 색상 구분으로 효율성 증대**
- 트랙 헤더 우클릭 → Customize → 색상 지정
- 권장 구분:
  - A1-A2 (대사): 파란색
  - A3-A4 (BGM): 초록색
  - A5-A6 (효과음): 노란색
  - A7-A8 (환경음): 보라색
- 복잡한 프로젝트에서 트랙 식별 속도 3배 향상
- 색상 통일로 팀 작업 시 협업 효율 증가

**팁 2: Ducking 키프레임을 수동으로 미세 조정**
- Auto-Ducking으로 80% 완성 후 수동 보완
- 방법:
  1. Essential Sound로 기본 덕킹 설정
  2. Effect Controls → Volume 키프레임 직접 수정
  3. 대사 시작 0.5초 전부터 미리 낮추기 (자연스러운 전환)
  4. 대사 끝나고 0.5초 후 복구 (여유 있는 느낌)
- 특히 빠른 대화나 랩 음악에서 효과적
- **시간 투자**: 초벌 5분 + 미세 조정 3분 = 총 8분

**팁 3: 주파수 충돌 방지 (EQ 활용)**
- 대사와 BGM의 주파수 대역 분리
- 방법:
  1. BGM 클립 선택 → Essential Sound → Music
  2. EQ 섹션 → Low Cut: 100-150Hz (대사 주파수 비우기)
  3. 대사 클립 → Essential Sound → Dialogue
  4. Clarity → EQ 활성화 (음성 주파수 강조)
- 결과: 대사가 더 선명하게 들리고 BGM과 분리됨
- **프로 기준**: 음성은 300-3000Hz, BGM은 그 외 대역 활용

**팁 4: 오디오 파형으로 편집점 빠르게 찾기**
- 타임라인에서 파형 크기 확대: 트랙 높이 드래그
- 시각적으로 대사 시작/끝, 호흡, 침묵 구간 즉시 파악
- 무음 구간 (파형이 평평한 곳) 찾아 컷 편집
- **편집 속도**: 파형 보고 편집 시 2-3배 빠름
- 추가 팁: 파형 색상도 변경 가능 (Preferences → Timeline → Audio Waveform Color)

**팁 5: 마스터 트랙에 Limiter 적용으로 안전장치**
- 최종 안전망으로 마스터 트랙에 Limiter 설치
- 방법:
  1. Audio Track Mixer 열기 (Window → Audio Track Mixer)
  2. Master 트랙 선택
  3. Effects and Sends → Hard Limiter 드래그
  4. Max Threshold: -1.0dB 설정
- 혹시 모를 클리핑을 자동으로 방지
- **주의**: Limiter는 최후의 수단, 개별 클립 조정이 우선

**팁 6: 참조 트랙(Reference Track) 활용**
- 좋아하는 유튜버나 영화의 오디오를 참조 트랙으로 임포트
- 자신의 편집과 비교하며 음량 밸런스 확인
- 특히 처음 시작할 때 유용 (감각 형성)
- A/B 비교로 객관적 평가 가능
- **방법**: 참조 트랙을 A8에 배치, 음소거 상태로 두고 필요 시 켜서 비교

**팁 7: 오디오 전용 시퀀스로 믹싱 집중**
- 복잡한 프로젝트에서 오디오만 따로 믹싱
- 방법:
  1. 기존 시퀀스 복제 (Duplicate)
  2. 이름: "Audio Mix Only"
  3. 비디오 트랙 모두 숨기기 (눈 아이콘)
  4. Audio Track Mixer에 집중하며 믹싱
  5. 완료 후 원본 시퀀스로 복사
- 시각적 방해 없이 청각에만 집중 가능
- **효율**: 믹싱 품질 20-30% 향상

#### 5.16 실전 예시: 팟캐스트 오디오 믹싱 완성

**시나리오**: 두 명의 호스트가 대화하는 30분 팟캐스트 에피소드 믹싱

**준비 단계 (2분)**
1. 프로젝트 열기 및 오디오 파일 확인
   - 호스트 1: 외장 마이크 녹음 (Host1.wav)
   - 호스트 2: 외장 마이크 녹음 (Host2.wav)
   - 원격 게스트: Zoom 녹음 (Guest_Zoom.wav) - 음질 낮음
2. 시퀀스 생성: 오디오 전용 (Video 트랙 불필요)
3. 트랙 배치:
   - A1: Host1
   - A2: Host2
   - A3: Guest (Zoom)
   - A4-A5: BGM (인트로, 아웃트로)
   - A6: 효과음 (전환 구간)

**1단계: 기본 음량 정규화 (5분)**
- 목표: 모든 호스트의 음량을 동일하게
- Host1 선택:
  1. Essential Sound → Dialogue 지정
  2. Loudness → Auto-Match 체크
  3. Target Loudness: -16 LUFS (팟캐스트 표준)
- Host2도 동일 작업
- Guest 오디오:
  1. 음량이 낮으면 Volume Level +5dB ~ +8dB 먼저 조정
  2. 그 후 Auto-Match 적용
- **결과 확인**: Audio Meter로 모든 대사가 -12dB ~ -6dB 범위 확인

**2단계: 노이즈 제거 및 음성 개선 (8분)**
- Host1, Host2 처리:
  1. Essential Sound → Repair
  2. Reduce Noise: 25% (에어컨 소음 제거)
  3. Reduce Rumble: 체크 (저음 잡음 제거)
  4. DeHum: 60Hz (전기 험 제거, 필요 시)
- Guest (Zoom 오디오) 특별 처리:
  1. Reduce Noise: 35% (인터넷 노이즈 많음)
  2. DeReverb: 30% (방 울림 제거)
  3. Clarity → Dynamics: 5-7 (음성 선명도 증가)
  4. EQ: High Pass 80Hz (저음 제거)
- **A/B 비교**: 전/후 재생하며 과도한 처리 방지

**3단계: 다이나믹 레인지 압축 (4분)**
- 목적: 작은 소리는 키우고, 큰 소리는 줄여서 일정하게
- 모든 대사 트랙에 적용:
  1. Essential Sound → Dynamics
  2. Amount: 5-7 (팟캐스트는 높게 설정)
- 또는 수동 Compressor:
  1. Audio Effects → Dynamics → Multiband Compressor
  2. Preset: "Broadcast" 선택
  3. Threshold: -18dB, Ratio: 3:1
- **효과**: 듣기 편한 일관된 음량, 피로도 감소

**4단계: 인트로/아웃트로 음악 추가 (6분)**
- 인트로 (15초):
  1. BGM 파일을 A4 트랙 시작 부분 배치
  2. 음량: -18dB (대사 없으므로 약간 높게)
  3. 길이 조정: 정확히 15초로 자르기
  4. 페이드 아웃: 마지막 2초에 Constant Power 적용
- 대사 시작 지점 (15초부터):
  1. BGM을 A5 트랙으로 계속 연장
  2. 음량 즉시 -25dB로 낮춤 (대사 방해 안 되게)
  3. 또는 Essential Sound → Ducking 설정:
     - Duck Against: A1, A2, A3 (모든 대사 트랙)
     - Duck Amount: 25dB
     - Sensitivity: High
- 아웃트로 (마지막 20초):
  1. 대사 끝나는 지점부터 BGM 음량 -18dB로 복구
  2. 페이드 아웃: 마지막 3초에 Constant Power

**5단계: 호흡 소리 및 불필요한 소음 제거 (7분)**
- 재생하며 다음 소리 찾기:
  - 큰 호흡 소리
  - 입 소리 (침 삼키는 소리)
  - 키보드 타이핑
  - 기침, 코 훌쩍이는 소리
- 처리 방법:
  1. Razor Tool (C)로 해당 구간 분할
  2. 음량을 -∞dB로 낮춤 (완전 무음)
  3. 또는 구간 삭제 후 Ripple Delete
- **주의**: 0.2-0.5초 이내의 짧은 침묵은 자연스러움
- **팁**: 파형 크게 확대해서 작업 (시각적으로 쉽게 찾기)

**6단계: 크로스토크 및 겹치는 대화 정리 (5분)**
- 두 사람이 동시에 말하는 구간:
  1. 파형으로 누가 주 발언자인지 확인
  2. 배경 음성(다른 호스트)의 음량 살짝 낮춤 (-3dB)
  3. 또는 Side-Chain Compression으로 자동 처리
- **자연스러운 대화 흐름 유지가 중요**

**7단계: 효과음 추가 (선택사항, 3분)**
- 토픽 전환 지점에 효과음:
  1. "transition whoosh" 효과음을 A6 트랙에 배치
  2. 음량: -12dB
  3. 길이: 0.5-1초
- 중요 포인트 강조:
  1. "notification" 효과음
  2. 음량: -10dB
- **주의**: 과도하게 사용 금지 (3-5회 이내)

**8단계: 최종 믹싱 및 마스터링 (5분)**
- Audio Track Mixer 열기:
  1. Master 트랙에 Hard Limiter 적용
  2. Max Threshold: -1.0dB
  3. Release Time: 100ms
- 전체 재생하며 최종 확인:
  - Audio Meter가 빨간색 안 들어가는지
  - 모든 대사가 명확하게 들리는지
  - BGM이 대사를 방해하지 않는지
- 헤드폰과 스피커 둘 다로 확인

**9단계: 내보내기 (2분)**
- File → Export → Media
- Format: WAV (무손실) 또는 MP3 (배포용)
- Audio 설정:
  - Sample Rate: 48000 Hz
  - Channels: Stereo
  - Bitrate: 320 kbps (MP3인 경우)
- Export

**총 소요 시간: 약 45-50분 (30분 콘텐츠 기준)**

**체크리스트**
- [ ] 모든 대사 음량 정규화 (-16 LUFS)
- [ ] 노이즈 제거 및 음성 개선
- [ ] 다이나믹 레인지 압축
- [ ] BGM 추가 및 덕킹 설정
- [ ] 호흡/잡음 제거
- [ ] 크로스토크 정리

- [ ] 클리핑 없음 확인
- [ ] 다중 기기로 청취 테스트

#### 5.17 고급 오디오 기법

**기법 1: Side-Chain Compression (사이드체인 압축)**
- 자동 덕킹의 고급 버전
- BGM이 대사에 반응하여 자동으로 음량 조절
- **설정 방법**:
  1. BGM 트랙 선택
  2. Audio Effects → Dynamics → Dynamics 이펙트 적용
  3. Sidechain → Key Input: A1 (대사 트랙)
  4. Threshold: -20dB, Ratio: 4:1, Attack: 10ms, Release: 200ms
- **효과**: Essential Sound Ducking보다 더 자연스럽고 정밀한 제어
- **활용**: 영화 예고편, 음악 비디오, 다큐멘터리

**기법 2: 공간감 연출 (Panning & Reverb)**
- 오디오에 입체감과 깊이 추가
- **Panning (좌우 배치)**:
  1. Audio Clip Mixer 또는 Audio Track Mixer 열기
  2. Pan 슬라이더로 소리를 좌/우로 이동
  3. 예시: 인터뷰에서 호스트는 좌측(-30), 게스트는 우측(+30)
- **Reverb (잔향)**:
  1. Audio Effects → Reverb → Studio Reverb
  2. Wet: 15-25% (자연스러운 공간감)
  3. Decay Time: 0.8-1.5초 (실내 느낌)
- **주의**: 과도한 리버브는 음성을 흐리게 함 (뉴스, 교육은 리버브 최소)
- **활용 사례**: 음악 공연, 영화적 분위기, ASMR

**기법 3: 오디오 키프레임 자동화**
- 복잡한 음량 변화를 키프레임으로 자동화
- **방법**:
  1. 클립 선택 → Effect Controls
  2. Volume → Level 스톱워치 클릭
  3. 재생 헤드를 원하는 지점마다 이동
  4. Level 값 조정 (자동으로 키프레임 생성)
  5. 키프레임 사이는 자동으로 보간(interpolation)
- **곡선 조정**:
  - 키프레임 우클릭 → Bezier 선택
  - 핸들 드래그로 부드러운 곡선 생성
- **활용**: 음악의 크레센도, 긴장감 조성, 배경음 페이드

**기법 4: 멀티밴드 압축기로 주파수별 제어**
- 저음, 중음, 고음을 각각 다르게 압축
- **사용법**:
  1. Audio Effects → Multiband Compressor 적용
  2. Preset: "Broadcast" 또는 "Voice Over"
  3. 각 밴드(Low, Mid, High) 개별 조정:
     - Low Band: 저음 (20-250Hz) - 럼블 제거
     - Mid Band: 음성 (250-5000Hz) - 압축 강화
     - High Band: 고음 (5000-20000Hz) - 선명도 유지
- **세부 설정**:
  - Threshold: -15dB, Ratio: 3:1
  - Attack: 5-10ms, Release: 50-100ms
- **효과**: 더욱 전문적이고 방송 품질의 오디오
- **활용**: 팟캐스트, 유튜브 보이스오버, 온라인 강의

**기법 5: 노이즈 프로파일 학습 (Adobe Audition 연동)**
- Premiere Pro 기본 노이즈 제거보다 강력
- **워크플로우**:
  1. Premiere에서 클립 우클릭 → Edit Clip in Adobe Audition
  2. Audition 자동 실행 및 클립 열림
  3. Effects → Noise Reduction / Restoration → Noise Reduction (process)
  4. 노이즈만 있는 구간 선택 (대사 없는 부분)
  5. "Capture Noise Print" 클릭
  6. 전체 클립 선택 (Ctrl + A)
  7. Noise Reduction 슬라이더 조정 (50-70%)
  8. Apply → Save → Premiere에 자동 반영
- **장점**: 복잡한 배경 소음도 효과적 제거
- **활용**: 시끄러운 환경 촬영, 바람 소리, 교통 소음

**기법 6: 오디오 시퀀스 네스팅**
- 복잡한 오디오 편집을 하나의 클립으로 그룹화
- **방법**:
  1. 여러 오디오 클립 선택
  2. 우클릭 → Nest
  3. 네스트된 시퀀스 이름 입력
  4. 하나의 클립처럼 작동 (이동, 복사, 이펙트 적용)
- **장점**:
  - 타임라인 정리
  - 전체 그룹에 이펙트 일괄 적용
  - 버전 관리 용이 (네스트 더블클릭으로 내부 편집)
- **활용**: 복잡한 효과음 레이어, 음악 스템 믹스, 재사용 가능한 오디오 템플릿

**기법 7: 참조 레벨 모니터링 (Loudness Radar)**
- 방송/스트리밍 플랫폼 표준에 맞춰 음량 최적화
- **설정**:
  1. Window → Loudness Radar
  2. Target Loudness:
     - YouTube: -14 LUFS
     - Spotify: -14 LUFS
     - Apple Podcasts: -16 LUFS
     - Broadcast TV: -24 LUFS
  3. 실시간 재생하며 레이더 모니터
  4. 초록색 영역 유지가 목표
- **조정**:
  - 레이더가 너무 낮으면: 전체 음량 +2dB
  - 레이더가 너무 높으면: 전체 음량 -2dB 또는 Limiter 강화
- **효과**: 플랫폼 자동 정규화로 인한 음질 저하 방지

#### 5.18 문제 해결

**문제 1: 오디오가 한쪽에서만 들림 (모노/스테레오 문제)**
- **증상**: 헤드폰으로 들으면 좌측 또는 우측에서만 소리 나옴
- **원인**: 모노 소스를 스테레오 트랙에 잘못 매핑
- **해결**:
  - 클립 우클릭 → Modify → Audio Channels
  - Clip Channel Format: Mono 선택
  - Source Channel: Channel 1
  - OK → 이제 양쪽에서 들림
- **예방**: 임포트 시 오디오 채널 자동 감지 확인

**문제 2: 오디오와 비디오 싱크가 점점 어긋남 (드리프트)**
- **증상**: 처음엔 맞는데 시간이 지날수록 점점 벌어짐
- **원인**: 카메라와 레코더의 샘플 레이트 불일치
  - 카메라: 48kHz
  - 레코더: 44.1kHz
- **해결**:
  1. 오디오 클립 우클릭 → Modify → Audio Channels
  2. Sample Rate를 시퀀스와 동일하게 설정 (48000Hz)
  3. 또는 Adobe Audition에서 Sample Rate Conversion
- **근본 해결**: 촬영 전 모든 기기를 48kHz로 통일

**문제 3: Essential Sound 패널이 비활성화됨**
- **증상**: Essential Sound 옵션이 회색으로 표시되어 클릭 불가
- **원인**: 오디오 클립이 선택되지 않았거나 비디오 클립 선택됨
- **해결**:
  - 타임라인에서 오디오 클립만 선택
  - Audio 트랙의 클립인지 확인 (Video 트랙 아님)
  - 클립을 한 번 더 클릭하여 활성화
- **확인**: Effect Controls 패널에 오디오 속성이 보이는지 확인

**문제 4: Ducking이 작동하지 않음**
- **증상**: Auto-Ducking 설정했는데 BGM 음량이 안 줄어듦
- **원인**: 대사 트랙 지정 오류 또는 Sensitivity 설정 문제
- **해결**:
  1. BGM 클립 재선택 → Essential Sound → Music
  2. Ducking → Duck Against에서 정확한 대사 트랙 선택 (A1, A2 등)
  3. Sensitivity를 High로 변경 (작은 소리에도 반응)
  4. Duck Amount를 20dB 이상으로 설정 (효과 명확)
- **테스트**: 대사 구간과 무음 구간 재생하며 BGM 파형 변화 확인

**문제 5: 오디오 재생 시 끊김 현상**
- **증상**: 재생 중 오디오가 뚝뚝 끊기거나 딜레이 발생
- **원인**:
  - 많은 이펙트로 CPU 과부하
  - 고해상도 비디오와 동시 재생
  - 버퍼 사이즈 부족
- **해결**:
  1. Edit → Preferences → Audio Hardware
  2. Buffer Size를 1024 또는 2048로 증가 (지연 시간은 늘지만 안정성 향상)
  3. 오디오 트랙 우클릭 → Render and Replace (오디오만)
  4. 또는 Sequence → Render Audio
- **임시 방법**: 재생 해상도를 1/4로 낮춤

**문제 6: 내보낸 영상에 오디오가 없음**
- **증상**: Export 후 영상은 있는데 소리가 안 나옴
- **원인**:
  - Export 설정에서 Audio 체크 해제
  - 오디오 트랙이 음소거 또는 비활성화됨
- **해결**:
  1. File → Export → Media
  2. Export Settings → Audio 탭 확인
  3. "Export Audio" 체크박스 확인
  4. Audio Codec: AAC 선택
  5. Bitrate: 320 kbps
  6. Sample Rate: 48000 Hz
- **타임라인 확인**: 오디오 트랙의 "M" (Mute) 버튼이 꺼져 있는지, 눈 아이콘이 켜져 있는지 확인

**문제 7: 오디오가 너무 작게 내보내짐**
- **증상**: 편집 중엔 잘 들렸는데 내보낸 파일은 소리가 작음
- **원인**: Master 음량이 낮게 설정됨 또는 과도한 Limiter
- **해결**:
  1. Audio Track Mixer에서 Master 레벨 확인
  2. 0dB에 맞춰져 있는지 확인
  3. Loudness Radar로 LUFS 측정:
     - 너무 낮으면 (-18 LUFS 이하): 전체 클립 선택 → Volume +3dB
  4. 재내보내기 전 Audio Meter로 피크 레벨 확인 (-3dB ~ -1dB)
- **예방**: 내보내기 전 다양한 기기(폰, 노트북, 헤드폰)에서 테스트

#### 5.19 연습 과제

**초급 과제 1: BGM 페이드 완벽 마스터**
- **난이도**: ⭐
- **목표**: 3개의 BGM을 자연스럽게 전환
- **준비물**: 영상 클립 1개, BGM 3개 (각 다른 분위기)
- **작업**:
  1. 첫 BGM: 0-30초, 페이드 인 2초, 페이드 아웃 3초
  2. 둘째 BGM: 30-60초, 크로스페이드 2초
  3. 셋째 BGM: 60-90초, 페이드 아웃 4초
  4. 각 BGM의 음량을 -20dB로 통일
- **평가 기준**:
  - [ ] 페이드가 부드럽고 자연스러움
  - [ ] BGM 교체 지점이 눈에 띄지 않음
  - [ ] 음량이 일정함 (클리핑 없음)
- **시간 제한**: 15분

**초급 과제 2: 노이즈 제거 Before/After**
- **난이도**: ⭐
- **목표**: 노이즈가 있는 대사를 깨끗하게 만들기
- **준비물**: 배경 소음이 있는 인터뷰 영상
- **작업**:
  1. Essential Sound → Dialogue 지정
  2. Repair → Reduce Noise 적용 (20-35%)
  3. Reduce Rumble 체크
  4. Before/After 비교 (이펙트 on/off)
- **평가 기준**:
  - [ ] 배경 소음이 줄어듦
  - [ ] 음성이 로봇처럼 들리지 않음
  - [ ] 명료도 향상
- **시간 제한**: 10분

**중급 과제 3: 팟캐스트 10분 에피소드 믹싱**
- **난이도**: ⭐⭐
- **목표**: 두 호스트의 대화를 전문적으로 믹싱
- **준비물**: 호스트 2명 오디오, BGM, 효과음
- **작업**:
  1. 음량 정규화 (-16 LUFS)
  2. 노이즈 제거 및 EQ 적용
  3. BGM 추가 및 Auto-Ducking 설정
  4. 호흡/잡음 제거
  5. 효과음 3-5개 추가 (전환 구간)
  6. 최종 Limiter 적용
- **평가 기준**:
  - [ ] 두 호스트 음량 균일
  - [ ] BGM이 대사를 방해하지 않음
  - [ ] 노이즈가 효과적으로 제거됨
  - [ ] 전문적인 완성도
- **시간 제한**: 40분

**고급 과제 4: 영화 장면 5.1 서라운드 믹스 (도전)**
- **난이도**: ⭐⭐⭐
- **목표**: 스테레오를 넘어 서라운드 사운드 경험
- **준비물**: 액션 장면 영상, 여러 효과음
- **작업**:
  1. 시퀀스 설정을 5.1 Surround로 변경
  2. 대사: Center 채널
  3. 배경음악: Front L/R
  4. 효과음: 위치에 따라 Panning
  5. Surround 채널에 환경음 배치
  6. Audio Track Mixer에서 5.1 밸런스 조정
- **평가 기준**:
  - [ ] 공간감이 느껴짐
  - [ ] 효과음이 화면 위치와 일치
  - [ ] 전문적인 서라운드 믹스
- **시간 제한**: 60분
- **참고**: 5.1 헤드폰 또는 서라운드 시스템 권장
