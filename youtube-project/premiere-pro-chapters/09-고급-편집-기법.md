### Chapter 9: 고급 편집 기법
**⭐ 랭킹 쇼츠 재편집 핵심 챕터**

#### 9.1 마스킹 (Masking)

**마스크란?**
- 영상의 특정 부분만 보이거나 숨기기
- 크롭과 비슷하지만 더 정밀
- 복잡한 모양 가능

**마스크 사용 사례**
- 얼굴 모자이크
- 특정 영역만 블러
- 화면 분할
- **랭킹 쇼츠**: 영상 특정 부분만 표시

**마스크 생성 방법**

**1. Opacity를 통한 마스크**
1. 클립 선택
2. Effect Controls → Opacity 섹션 펼치기
3. 도구 선택:
   - **Free Draw Bezier**: 자유로운 곡선
   - **Ellipse**: 원형
   - **Rectangle**: 사각형
4. 프로그램 모니터에서 마스크 그리기
5. Mask Path로 위치/크기 조정

**2. 마스크 반전**
- Inverted 체크: 마스크 안쪽이 아닌 바깥쪽 적용
- 용도: 특정 부분 외 나머지 블러

**3. Feather (페더)**
- 마스크 경계 부드럽게
- 수치: 50-100 (자연스러움)

#### 9.2 크롭 (Crop) 고급 활용

**Crop 이펙트 (Transform과 다름)**
- Video Effects → Transform → Crop
- 클립에 드래그

**크롭 설정**
- **Left, Right, Top, Bottom**: 각 방향 크롭 %
- **Feather**: 경계 부드럽게

**랭킹 쇼츠 활용**
- 가로 영상을 세로 화면에 맞추기
- 상하 또는 좌우 불필요한 부분 제거
- Top/Bottom 30-40% 크롭으로 핵심만 표시

**화면 분할 (Split Screen)**
1. V1에 첫 번째 클립 배치
2. V2에 두 번째 클립 배치
3. V2 클립에 Crop 적용:
   - Left: 50% (오른쪽 절반만 표시)
4. V1 클립에 Crop 적용:
   - Right: 50% (왼쪽 절반만 표시)
5. 화면 2분할 완성

#### 9.3 모션 트래킹 (Motion Tracking)

**모션 트래킹이란?**
- 움직이는 대상을 자동으로 추적
- 텍스트나 그래픽이 대상을 따라 이동

**사용 예시**
- 제품에 화살표 또는 텍스트 고정
- 얼굴 모자이크가 자동으로 따라가기
- 움직이는 차량에 정보 표시

**트래킹 방법**

**1. 트랙할 대상 준비**
1. 클립 선택
2. Effect Controls → Track Motion (또는 Effects → Track Motion)

**2. 트래킹 설정**
1. Track Type:
   - **Position**: 위치만 추적
   - **Position, Scale, Rotation**: 크기 및 회전도 추적
2. 트래킹 박스를 추적할 대상에 맞춤
3. Play 버튼 → 자동 추적 시작
4. 추적 완료 후 "Apply"

**3. 텍스트/그래픽 연결**
1. 텍스트 또는 그래픽 레이어 생성
2. Track Motion 데이터를 해당 레이어에 적용
3. 자동으로 대상을 따라 이동

#### 9.4 키잉 (Keying) - 그린스크린

**키잉이란?**
- 특정 색상(보통 그린/블루) 제거
- 배경 교체
- 합성 영상 제작

**Ultra Key 사용법**
1. 그린스크린 클립 V2 트랙에 배치
2. 배경 영상 V1 트랙에 배치
3. V2 클립 선택
4. Effects → Video Effects → Keying → Ultra Key
5. Effect Controls → Ultra Key
6. Eye Dropper로 그린 배경 클릭
7. 자동으로 그린 제거

**세부 조정**
- **Matte Generation → Transparency**: 투명도 조정
- **Matte Cleanup → Choke**: 경계선 정리
- **Spill Suppression → Desaturate**: 그린 반사광 제거

#### 9.5 고급 속도 조절

**Optical Flow (광학 흐름)**
- 더 부드러운 슬로우 모션
- 프레임 사이 보간

**사용 방법**
1. 클립 우클릭 → Speed/Duration
2. Speed: 50% (또는 원하는 속도)
3. Time Interpolation: **Optical Flow**
4. OK → 렌더링 대기

**주의**
- 렌더링 시간 오래 걸림
- 움직임이 복잡하면 아티팩트 발생 가능

**Time Remapping (고급 속도 제어)**
1. 클립 우클릭 → Show Clip Keyframes → Time Remapping → Speed
2. Ctrl + 클릭으로 속도 변경 지점 추가
3. 키프레임 드래그하여 속도 조절
   - 위로: 빠르게
   - 아래로: 느리게
4. 부드러운 전환: 키프레임 우클릭 → Ease In/Ease Out

#### 9.6 Adjustment Layer (조정 레이어)

**조정 레이어란?**
- 투명한 레이어에 이펙트 적용
- 아래 모든 트랙에 영향
- 여러 클립에 동시 효과 적용

**사용 방법**
1. File → New → Adjustment Layer
2. 타임라인의 최상위 트랙에 배치 (V3 이상)
3. 조정 레이어에 이펙트 적용 (예: Lumetri Color)
4. 아래 모든 클립에 동시 적용

**활용 사례**
- 전체 씬에 컬러 그레이딩
- 화면 전체에 비네팅
- 일괄 밝기 조정

#### 9.7 블렌드 모드 (Blend Mode)

**블렌드 모드란?**
- 레이어 간 혼합 방식
- Photoshop과 동일 개념

**사용 방법**
1. 위쪽 트랙 클립 선택
2. Effect Controls → Opacity → Blend Mode
3. 모드 선택

**주요 블렌드 모드**
- **Multiply**: 어두운 부분 강조
- **Screen**: 밝은 부분 강조
- **Overlay**: 대비 증가
- **Add**: 빛 효과 (밝아짐)

**활용 예시**
- 라이트 누출(Light Leak) 효과
- 텍스처 오버레이
- 빈티지 필름 느낌

#### 9.8 네스트 시퀀스 (Nest Sequence)

**네스트란?**
- 여러 클립을 하나의 시퀀스로 묶기
- 그룹화하여 관리
- 일괄 효과 적용

**사용 방법**
1. 여러 클립 선택
2. 우클릭 → Nest
3. 이름 입력 → OK
4. 하나의 클립처럼 취급

**활용**
- 복잡한 편집 단순화
- 네스트 전체에 이펙트 한 번에 적용
- 타임라인 정리

#### 9.9 랭킹 쇼츠 재편집 실전

- 1080x1920 (9:16) 세로 시퀀스 생성

**2. 원본 영상 배치**
- 가로 영상 클립을 세로 시퀀스에 배치

**5. 순위 자막 추가**
- 큰 폰트로 "#1", "#2", "#3" 표시
- 금색, 은색, 동색으로 차별화
- 위치: 좌측 상단 또는 중앙

**6. 설명 텍스트**
- 작은 폰트로 설명 추가
- 우측 또는 하단 배치

**7. 마스킹 (선택)**
- 원본 영상 특정 부분만 하이라이트
- 나머지는 약간 어둡게 또는 블러

#### 9.11 자주 하는 실수와 해결법

**실수 1: 마스크가 클립을 따라가지 않음 (고정된 위치)**
- **원인**: 마스크를 클립 자체가 아닌 Opacity에 적용
- **해결**:
  - 움직이는 대상: Motion Tracking 사용 필수
  - Effect Controls → Track Motion 활성화
  - 추적 완료 후 마스크 데이터 연결
- **대안**: 매 프레임마다 키프레임 수동 조정 (비효율적)

**실수 2: 크롭 적용 후 화질이 깨짐**
- **원인**: 과도한 Scale 확대
- **해결**:
  - 원본 영상이 고해상도인지 확인 (4K 권장)
  - Scale 150% 이상은 화질 저하 불가피
  - 크롭 대신 시퀀스 설정을 영상에 맞춤
- **예방**: 촬영 단계에서 구도 미리 계획

**실수 3: Ultra Key 그린스크린 제거 시 경계가 거칠음**
- **원인**: 조명 불균일, 세부 조정 부족
- **해결**:
  - Matte Cleanup → Choke: 1-3% (경계선 다듬기)
  - Pedestal: 10 (어두운 부분 제거)
  - Spill Suppression → Desaturate: 50% (그린 반사광 제거)
- **촬영 팁**: 그린스크린과 피사체 사이 2m 이상 거리

**실수 4: Optical Flow 슬로우 모션에 아티팩트 발생**
- **원인**: 움직임이 너무 복잡하거나 빠름
- **해결**:
  - Frame Sampling 또는 Frame Blending으로 변경
  - 또는 원본을 60fps/120fps로 촬영
  - Optical Flow는 단순한 움직임에만 효과적
- **대안**: After Effects의 Twixtor 플러그인 (유료)

**실수 5: 조정 레이어가 의도하지 않은 클립에도 적용됨**
- **원인**: 조정 레이어가 최상위 트랙에 있어 모든 하위 트랙 영향
- **해결**:
  - 조정 레이어 위치 조정 (특정 구간만)
  - 또는 클립 길이 조절하여 필요한 부분만 커버
  - 네스트 시퀀스로 특정 클립만 묶어 적용
- **확인**: 타임라인에서 조정 레이어 범위 시각적으로 점검

**실수 6: 네스트 시퀀스 편집 후 원본 클립 수정이 어려움**
- **원인**: 네스트 내부 구조를 잊음
- **해결**:
  - 네스트 더블클릭하여 내부 시퀀스 열기
  - 수정 후 원래 시퀀스로 복귀 (탭 클릭)
  - 또는 네스트 우클릭 → "Reveal in Project" → 원본 시퀀스 확인
- **관리 팁**: 네스트 시퀀스에 명확한 이름 부여

**실수 7: 블렌드 모드가 제대로 작동하지 않음**
- **원인**: 하위 레이어가 없거나 불투명도 문제
- **해결**:
  - V1에 배경 레이어 있는지 확인
  - 블렌드 모드 적용 레이어의 Opacity 100% 확인
  - 다양한 블렌드 모드 실험 (Screen, Multiply, Overlay 등)
- **팁**: 레이어 순서 바꿔보기 (효과 달라짐)

#### 9.12 프로 팁 (고급 편집 가속화)

**팁 1: 마스크 페더 키프레임으로 동적 효과**
- Mask Feather 값에 키프레임 적용
- 시간에 따라 블러 강도 변경
- **사용 예**: 인물 등장 시 페더 0 → 등장 완료 시 페더 100
- 부드러운 전환 효과 연출

**팁 2: 여러 마스크 조합으로 복잡한 모양 생성**
- 하나의 클립에 여러 마스크 적용 가능
- 각 마스크마다 Add/Subtract/Intersect 모드 선택
- **예시**: 원형 마스크 2개 겹쳐 하트 모양 생성
- 창의적 비주얼 효과

**팁 3: 모션 트래킹 데이터 재사용**
- 한 번 추적한 데이터를 여러 레이어에 적용
- 트래킹 완료 후 Copy
- 다른 텍스트/그래픽에 Paste
- **효율**: 같은 대상 여러 요소 추가 시 시간 절약 90%

**팁 4: Adjustment Layer 일시 비활성화**
- 조정 레이어 왼쪽 눈 아이콘 클릭
- 효과 적용 전/후 빠른 비교
- 렌더링 시간 절약 (테스트 중)
- **활용**: 최종 확인 시에만 활성화

**팁 5: 크롭과 Position 조합으로 Ken Burns 효과**
- Ken Burns: 정지 이미지에 움직임 부여
- 설정:
  1. 이미지 Scale: 120%
  2. Position 시작: X=0, Y=0
  3. Position 끝 (5초 후): X=100, Y=100
  4. 동시에 Crop Top: 0% → 10% 키프레임
- **결과**: 이미지가 천천히 이동하며 확대

**팁 6: 네스트 전 컬러 라벨로 표시**
- 네스트할 클립들에 동일 색상 라벨
- 시각적으로 그룹 확인
- 네스트 후에도 추적 용이
- **색상 규칙**: 빨강=메인, 파랑=B-Roll, 초록=그래픽

**팁 7: Time Remapping 속도 그래프 스무스하게**
- 속도 키프레임 우클릭 → Ease In/Ease Out
- 급격한 속도 변화 방지
- 부드러운 Speed Ramp 효과
- **비교**: Ease 없으면 딱딱함, Ease 있으면 영화 같음

#### 9.13 실전 예시: 프로덕트 하이라이트 영상

**시나리오**: 제품 소개 영상에 고급 기법 적용

**사용 기술 조합**
1. **모션 트래킹** (제품에 정보 텍스트 고정)
2. **마스킹** (배경 블러, 제품만 선명)
3. **Speed Ramp** (제품 등장 시 슬로우 모션)
4. **Adjustment Layer** (전체적인 컬러 톤 통일)
5. **네스트 시퀀스** (복잡한 합성 단순화)

**단계별 프로세스 (15분)**

**1단계: 기본 편집 (3분)**
- 제품 영상 클립 배치
- 불필요한 부분 컷
- 전체 길이 15초로 조정

**2단계: 마스킹으로 제품 강조 (4분)**
- 제품 클립 V2에 배치
- V1에 동일 클립 복제 (배경용)
- V1 클립에 Gaussian Blur (80)
- V2 클립에 마스크 그리기 (제품 영역)
- Feather: 50 (자연스러운 경계)
- **결과**: 제품만 선명, 배경 블러

**3단계: 모션 트래킹 (4분)**
- 제품 움직임 추적
- Track Type: Position
- 추적 완료 후 텍스트 레이어에 적용
- 제품명 텍스트가 제품을 따라 이동

**4단계: Speed Ramp (2분)**
- 제품 등장 구간 (2-4초) 선택
- Time Remapping으로 속도 50%로 감속
- Ease In/Out 적용
- **효과**: 제품 등장이 극적으로 보임

**5단계: 조정 레이어로 통일감 (2분)**
- 최상위 트랙에 Adjustment Layer
- Lumetri Color 적용
  - Temperature: +10 (따뜻한 톤)
  - Contrast: +20
  - Saturation: +10
- 전체 영상에 일관된 색감

**최종 결과**: 프로페셔널한 제품 영상 완성

#### 9.14 고급 컴포지팅 기법

**Picture-in-Picture (PIP) 고급 버전**
- 기본 PIP에 추가 효과:
  1. Scale & Position으로 작은 화면 배치
  2. Drop Shadow 이펙트로 입체감
  3. Crop 또는 마스크로 모서리 둥글게
  4. Transform → Bevel Edges (테두리 강조)
- **응용**: 리액션 영상, 게임+캠 화면

**Split Screen 3-4분할**
- 2분할 확장 버전
- **4분할 레이아웃**:
  - V4: 좌상단 (Crop: Right 50%, Bottom 50%)
  - V3: 우상단 (Crop: Left 50%, Bottom 50%)
  - V2: 좌하단 (Crop: Right 50%, Top 50%)
  - V1: 우하단 (Crop: Left 50%, Top 50%)
- Position 미세 조정으로 중앙 간격 조절
- **용도**: 비교 영상, 멀티캠

**마스크 애니메이션 (Wipe 전환)**
- 마스크를 전환 효과로 활용
- **원형 와이프**:
  1. 타원 마스크 생성
  2. Mask Expansion 키프레임: 0 → 500
  3. 장면이 원형으로 확장되며 전환
- **슬라이드 와이프**:
  1. 사각형 마스크
  2. Position 키프레임: 좌→우 이동
- **효과**: 독특한 전환 연출

**Track Matte (트랙 매트)**
- 한 레이어가 다른 레이어의 투명도 제어
- Premiere Pro는 제한적 지원
- **방법**:
  1. Effects → Set Matte
  2. Take Matte From: 특정 비디오 레이어
  3. Use For Matte: Luminance 또는 Alpha
- **용도**: 텍스트 내부에 영상 재생, 창의적 합성

#### 9.15 문제 해결 (고급 기법)

**문제 1: 모션 트래킹이 중간에 놓침**
- **원인**: 추적 대상이 가려지거나 화면 밖으로 나감
- **해결**:
  - 트래킹 일시 정지 → 수동으로 박스 재배치 → 재개
  - 또는 두 구간으로 나눠 트래킹 후 키프레임 연결
- **예방**: 촬영 시 대상 항상 프레임 안에 유지

**문제 2: 네스트 시퀀스가 원본 해상도와 다름**
- **원인**: 네스트 시퀀스 설정 불일치
- **해결**:
  - 네스트 우클릭 → Sequence Settings
  - 원본 시퀀스와 동일하게 설정 (해상도, 프레임 레이트)
- **자동 해결**: 클립 선택 → 우클릭 → "Nest" (자동으로 설정 일치)

**문제 3: 마스크 경로 키프레임이 너무 많아 관리 어려움**
- **원인**: 자동 트래킹 또는 과도한 수동 조정
- **해결**:
  - 불필요한 키프레임 삭제
  - 또는 키프레임 간격 늘리기 (Linear 보간)
  - 단순화: 주요 지점만 키프레임 남기기
- **대안**: After Effects에서 Mocha 사용 (더 정교함)

**문제 4: Adjustment Layer 효과가 약함**
- **원인**: 하위 클립에 이미 강한 효과 적용됨
- **해결**:
  - 조정 레이어 이펙트 수치 증가
  - 또는 하위 클립 이펙트 약화
  - 중첩 효과 확인 (과도하면 부자연스러움)
- **권장**: 조정 레이어는 미묘한 톤 조정용

**문제 5: 크롭 후 클립이 이상한 위치에 배치됨**
- **원인**: Anchor Point가 중앙이 아님
- **해결**:
  - Effect Controls → Anchor Point 재설정
  - 또는 Position 값 수동 조정
- **자동 중앙**: Transform 이펙트 사용 (Anchor Point 자동 중앙)
