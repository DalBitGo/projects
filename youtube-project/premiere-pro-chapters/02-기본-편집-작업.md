### Chapter 2: 기본 편집 작업

#### 2.1 미디어 임포트하기

**방법 1: 드래그 앤 드롭 (가장 간편)**
- 탐색기에서 파일을 직접 프로젝트 패널로 드래그
- 여러 파일 동시 선택 가능

**방법 2: File 메뉴 사용**
- File → Import (Ctrl + I)
- 파일 선택 후 "열기"
- 폴더 전체 임포트: Import 대화상자에서 폴더 선택

**방법 3: Media Browser 사용**
- Window → Media Browser
- 파일 시스템 탐색 및 미리보기
- 우클릭 → Import

**지원 파일 형식**
- **비디오**: MP4, MOV, AVI, MKV, M4V
- **오디오**: MP3, WAV, AAC, M4A
- **이미지**: JPG, PNG, PSD, AI, TIFF
- **시퀀스**: 이미지 시퀀스 (001.jpg, 002.jpg...)

**프로젝트 패널 정리**
- 빈(Bin) 생성: 프로젝트 패널 하단 폴더 아이콘 클릭
- 폴더 구조 예시:
  ```
  프로젝트 패널/
  ├── 01_영상소스
  ├── 02_오디오
  ├── 03_BGM
  ├── 04_이미지
  └── 05_시퀀스
  ```
- 메타데이터 활용: 이름, 라벨 컬러로 분류

#### 2.2 타임라인에 클립 배치하기

**클립을 타임라인에 추가하는 방법**

**방법 1: 드래그 앤 드롭**
- 프로젝트 패널에서 클립을 타임라인으로 드래그
- 비디오 트랙(V1)과 오디오 트랙(A1) 자동 배치
- 드래그 중 타임라인 확대/축소: + / - 키

**방법 2: 소스 모니터 활용 (정밀 편집)**
1. 클립을 더블클릭하여 소스 모니터에서 열기
2. `I` 키로 In 포인트, `O` 키로 Out 포인트 설정
3. 원하는 구간만 선택
4. 드래그 또는 단축키로 삽입:
   - `,` (쉼표): 삽입 (Insert) - 기존 클립을 밀어냄
   - `.` (마침표): 덮어쓰기 (Overwrite) - 기존 클립 위에 덮어씀

**트랙 이해하기**
- **V1, V2, V3...**: 비디오 트랙 (위쪽이 전면)
- **A1, A2, A3...**: 오디오 트랙
- 트랙 추가: 타임라인 빈 공간 우클릭 → Add Tracks
- 트랙 잠금: 자물쇠 아이콘 (실수 방지)
- 트랙 숨기기: 눈 아이콘 (렌더링에서 제외)

#### 2.3 클립 자르기 (Trimming)

**핵심 개념**
- 편집의 기본은 "자르기"
- 불필요한 부분 제거, 필요한 부분만 남기기
- 리듬과 템포 조절

**자르기 도구 및 방법**

**1. Razor Tool (면도날 도구) - C 키**
- 클립을 특정 지점에서 분할
- 사용법:
  1. `C` 키로 Razor Tool 활성화
  2. 자를 위치 클릭
  3. `V` 키로 Selection Tool로 복귀
- 팁: Shift 누른 채 클릭 시 모든 트랙 동시 분할

**2. Selection Tool (선택 도구) - V 키**
- 클립 끝부분 드래그하여 길이 조절
- 노란색 숫자: 조정 중인 프레임 수 표시
- Alt 키 누른 채 드래그: 오디오/비디오 따로 조정

**3. Ripple Edit Tool - B 키**
- 클립 자르면 뒤 클립들이 자동으로 당겨짐
- 갭(공백) 없는 편집에 유용

**주요 자르기 단축키**
- `Q`: 재생 헤드 위치에서 클립 앞부분 자르기 (Trim In)
- `W`: 재생 헤드 위치에서 클립 뒷부분 자르기 (Trim Out)
- `Ctrl + K`: 재생 헤드 위치에서 클립 분할 (Razor와 동일)
- `Ctrl + Shift + K`: 모든 트랙 동시 분할

**실전 워크플로우 예시**
1. 타임라인에 클립 배치
2. 재생하며 자를 위치 찾기 (스페이스바)
3. 재생 헤드를 정확한 위치에 이동 (←/→)
4. `Ctrl + K`로 분할
5. 불필요한 부분 선택 후 `Delete` 삭제

#### 2.4 클립 이동 및 재배치

**기본 이동**
- 클립 선택 후 드래그
- Ctrl + X (잘라내기) → Ctrl + V (붙여넣기)
- Snap 기능 (`S` 키): 클립이 다른 클립이나 재생 헤드에 자석처럼 붙음

**갭(공백) 제거**
- **방법 1**: 갭 우클릭 → Ripple Delete
- **방법 2**: 갭 선택 후 `Delete` 키
- Ripple Delete는 뒤 클립들을 자동으로 당김

**클립 복사**
- Alt + 드래그: 클립 복제
- Ctrl + C → Ctrl + V: 복사 및 붙여넣기

**다중 선택**
- Shift + 클릭: 여러 클립 선택
- 드래그로 영역 선택
- Ctrl + A: 타임라인 내 모든 클립 선택

#### 2.5 기본 편집 단축키 완전정복

**필수 편집 단축키 (숙달 필수!)**

**도구 선택**
- `V`: Selection Tool (기본 도구)
- `C`: Razor Tool (자르기)
- `B`: Ripple Edit Tool
- `N`: Rolling Edit Tool
- `H`: Hand Tool (타임라인 이동)
- `Z`: Zoom Tool

**재생 및 탐색**
- `스페이스바`: 재생/정지
- `L`: 재생 속도 증가
- `J`: 역재생
- `K`: 정지
- `→/←`: 다음/이전 프레임
- `Shift + →/←`: 5프레임 이동
- `Page Up/Down`: 클립 단위 이동
- `Home/End`: 타임라인 처음/끝

**마커**
- `M`: 마커 추가 (중요한 지점 표시)
- `Shift + M`: 다음 마커로 이동
- `Ctrl + Shift + M`: 이전 마커로 이동

**편집 작업**
- `Ctrl + K`: 클립 분할
- `Ctrl + Shift + K`: 모든 트랙 분할
- `Q`: Trim In (앞부분 자르기)
- `W`: Trim Out (뒷부분 자르기)
- `Delete`: 선택 클립 삭제 (갭 남김)
- `Backspace` (Ripple Delete): 삭제 후 갭 제거
- `Ctrl + Z`: 실행 취소
- `Ctrl + Shift + Z`: 다시 실행

**클립 배치**
- `,` (쉼표): Insert (삽입)
- `.` (마침표): Overwrite (덮어쓰기)
- `I`: In 포인트 설정
- `O`: Out 포인트 설정
- `X`: In-Out 포인트 사이 선택

**타임라인 탐색**
- `+/-`: 타임라인 확대/축소
- `\`: 전체 타임라인 보기
- `S`: Snap 켜기/끄기

#### 2.6 오디오와 비디오 분리/연결

**링크 해제 (Unlink)**
- 클립 우클릭 → Unlink
- 비디오와 오디오 따로 편집 가능
- 용도: 영상은 유지하고 오디오만 교체

**링크 재설정**
- 비디오와 오디오 클립 동시 선택
- 우클릭 → Link

**오디오만 삭제/비디오만 삭제**
- 클립 선택 → Shift + Delete
- 대화상자에서 Video 또는 Audio 선택

**실전 활용 시나리오**
1. **B-Roll 추가 시**
   - 인터뷰 오디오는 유지
   - 비디오만 B-Roll로 교체
   - 링크 해제 후 비디오 트랙만 삭제

2. **음악 비디오 편집**
   - 영상 클립의 오디오 삭제
   - 배경 음악 트랙만 유지
   - 립싱크 맞추기

3. **다국어 더빙**
   - 원본 오디오 분리
   - 새 언어 오디오 트랙 추가
   - 타이밍 동기화

#### 2.8 자주 하는 실수와 해결법

**실수 1: 클립을 자를 때 오디오/비디오가 따로 잘림**
- **원인**: 링크가 해제된 상태
- **해결**:
  - 비디오와 오디오 클립 동시 선택
  - 우클릭 → Link로 다시 연결
  - Shift + Ctrl + K로 모든 트랙 동시 분할
- **예방**: 링크 상태 항상 확인 (클립 테두리 연결선 확인)

**실수 2: 갭(공백)이 계속 생김**
- **원인**: Delete 키 사용 (Ripple Delete 아님)
- **해결**:
  - 갭 우클릭 → Ripple Delete
  - 또는 갭 선택 후 Backspace 키
- **프로 팁**: Ripple Edit Tool (B) 사용하면 자동으로 갭 제거

**실수 3: 클립을 이동했는데 오디오가 동기화되지 않음**
- **원인**: 오디오와 비디오 따로 이동
- **해결**:
  - Ctrl + Z로 실행 취소
  - 클립 전체 선택 후 이동
  - 또는 Link 상태 확인
- **동기화 재정렬**: 우클릭 → Synchronize

**실수 4: 편집 중 타임라인이 너무 복잡해짐**
- **원인**: 불필요한 트랙이 많거나 클립 정리 안 됨
- **해결**:
  - 사용하지 않는 트랙 삭제
  - 클립에 라벨 색상 적용 (시각적 구분)
  - 네스트 시퀀스로 그룹화 (Chapter 9)
  - 빈 트랙 숨기기 (눈 아이콘)

**실수 5: 재생 헤드가 클립에 안 붙음 (스냅 안 됨)**
- **원인**: Snap 기능 꺼짐
- **해결**: `S` 키로 Snap 활성화
- **확인**: 타임라인 상단 자석 아이콘 파란색 확인

**실수 6: 클립을 복사했는데 속성도 같이 복사됨**
- **상황**: 이펙트나 색보정이 적용된 클립 복제
- **해결 (속성만 복사 원할 때)**:
  - 원본 클립 선택 → 우클릭 → Copy
  - 대상 클립 선택 → 우클릭 → Paste Attributes
  - 원하는 속성만 선택
- **해결 (속성 제외 복제)**:
  - 원본 클립의 이펙트 제거 후 복제
  - 또는 새 클립으로 시작

**실수 7: Razor Tool에서 빠져나오지 못함**
- **증상**: 계속 자르기만 됨
- **해결**: `V` 키로 Selection Tool로 복귀
- **팁**: Razor Tool은 임시 사용, 즉시 V로 복귀 습관화

#### 2.9 프로 팁 (편집 속도 향상)

**팁 1: Three-Point Editing 마스터**
- 소스 모니터에서 In/Out 포인트 설정
- 타임라인에서 In 포인트만 설정
- `,` 또는 `.` 키로 정확히 삽입
- 효율: 드래그보다 3배 빠름

**팁 2: J-K-L 재생 제어 완벽 활용**
- `J`: 역재생 (한 번 더 누르면 2배속, 3배속)
- `K`: 정지
- `L`: 정재생 (한 번 더 누르면 2배속, 3배속)
- `J+K` 또는 `K+L` 동시 누름: 슬로우 모션 재생
- **활용**: 정확한 컷 지점 찾기에 최적

**팁 3: 키보드 왼손 배치 최적화**
- 왼손: Q, W, E (자르기), A, S, D (도구), Z, X, C (선택/자르기)
- 오른손: 마우스 또는 방향키
- 단축키 근처에 손 위치하면 편집 속도 2배 향상

**팁 4: 마커 활용으로 편집 포인트 미리 표시**
- 재생하며 `M` 키로 마커 추가
- 나중에 해당 지점에서 정확히 편집
- 마커에 메모 추가: 마커 더블클릭 → 코멘트 입력
- **용도**: 음악 비트, 중요 장면, 편집 예정 지점

**팁 5: Ripple Trim 단축키 (Q, W) 완전 숙달**
- `Q`: 재생 헤드 앞부분 제거 (빠른 인트로 컷)
- `W`: 재생 헤드 뒷부분 제거 (빠른 아웃트로 컷)
- 재생 중 즉시 사용 가능
- **워크플로우**: 스페이스바 재생 → Q/W로 즉시 컷 → 이어서 재생

**팁 6: 프록시 워크플로우 (저사양 PC)**
- 4K 영상은 편집 중 끊김 발생
- **해결**: Proxy 미디어 생성
  1. 클립 우클릭 → Proxy → Create Proxies
  2. Format: H.264, Preset: Low Resolution
  3. 편집은 프록시로, 내보내기는 원본 자동 사용
- 편집 속도 5-10배 향상

**팁 7: 패널 레이아웃 빠른 전환**
- 숫자 키패드로 작업 공간 전환 설정
- Keyboard Shortcuts → Panel → 숫자에 워크스페이스 할당
- 예: `1` = Editing, `2` = Color, `3` = Audio
- 원클릭으로 레이아웃 변경

#### 2.10 편집 워크플로우 비교 (초보 vs 프로)

**초보자 워크플로우 (비효율적)**
1. 클립을 타임라인에 전부 배치
2. 마우스로 하나씩 클릭하며 자르기
3. Razor Tool로 분할 → 마우스로 불필요한 부분 선택 → Delete
4. 갭 발생 → 마우스로 클립 드래그해서 메우기
5. 반복

**소요 시간**: 10분 분량 편집에 30-40분

**프로 워크플로우 (효율적)**
1. 소스 모니터에서 In/Out 포인트로 원하는 구간만 선택
2. `,` 또는 `.` 키로 타임라인에 정확히 배치
3. 재생하며 `Q`, `W` 키로 즉석 트림
4. Ripple Delete로 갭 자동 제거
5. 단축키 위주 작업

**소요 시간**: 10분 분량 편집에 10-15분

**핵심 차이점**
- 프로는 소스 모니터 활용 (미리 선별)
- 단축키 사용률 80% 이상
- 갭 발생 최소화 (Ripple 도구 활용)
- 마우스 사용 최소화

#### 2.11 실전 예시: 유튜브 쇼츠 빠른 편집

**시나리오**: 1분 분량 원본을 30초 쇼츠로 압축

**단계별 프로세스**

**1단계: 사전 시청 및 구간 표시 (2분)**
- 원본 클립을 소스 모니터에서 재생
- 재미있는 부분, 핵심 포인트에 마커 (`M`) 추가
- 5-7개 구간 표시

**2단계: 구간 추출 (3분)**
- 첫 번째 마커 위치로 이동 (`Shift + M`)
- `I` 키로 In 포인트, 몇 초 뒤 `O` 키로 Out 포인트
- `.` 키로 타임라인에 배치
- 다음 마커로 이동, 반복

**3단계: 타이밍 조정 (2분)**
- 전체 재생하며 리듬 확인
- 너무 긴 클립은 `Q`, `W` 키로 트림
- 총 길이 30초 내외로 조정
- 타임라인 하단 시간 표시 확인

**4단계: 갭 제거 및 정리 (1분)**
- Backspace로 모든 갭 제거
- 클립 간 간격 확인
- Snap 기능 (`S`) 활용해 딱 붙이기

**5단계: 속도 조정 (선택사항, 1분)**
- 지루한 부분: 우클릭 → Speed/Duration → 150%
- 강조 부분: 50% (슬로우 모션)

**총 소요 시간: 약 10분**

#### 2.12 문제 해결 (Troubleshooting)

**문제 1: 타임라인에 클립을 드롭할 수 없음**
- **원인**: 트랙 잠김 또는 시퀀스 없음
- **해결**:
  - 트랙 왼쪽 자물쇠 아이콘 확인 → 클릭하여 잠금 해제
  - 시퀀스가 없으면 먼저 시퀀스 생성
  - 타임라인 패널이 활성화되어 있는지 확인 (클릭)

**문제 2: 클립이 빨간색 선으로 표시됨**
- **원인**: 시퀀스 설정과 클립 설정 불일치
- **해결**:
  - Enter 키로 해당 구간 렌더링
  - 또는 타임라인 해상도를 1/4로 낮춤
  - 프록시 미디어 생성
- **예방**: 첫 클립으로 시퀀스 자동 생성

**문제 3: Ctrl + K로 자르기가 안 됨**
- **원인**: 타임라인이 선택되지 않음
- **해결**: 타임라인 패널을 클릭하여 활성화
- **확인**: 타임라인 패널 테두리 파란색 확인

**문제 4: 클립 이동 시 다른 클립을 덮어씀**
- **원인**: Insert 대신 Overwrite 모드
- **해결**:
  - Ripple Edit Tool (B) 사용
  - 또는 클립을 빈 트랙으로 먼저 이동
- **주의**: 드래그 시 위치 신중히 확인

**문제 5: 단축키가 작동하지 않음**
- **원인**: 다른 패널이 활성화됨
- **해결**:
  - 타임라인을 클릭하여 활성화
  - 또는 프로그램 모니터 클릭
- **확인**: 패널 테두리 색상 확인 (활성화 시 파란색)

**문제 6: 편집 중 프로젝트가 자주 크래시**
- **원인**: 메모리 부족, 캐시 누적
- **해결**:
  - Edit → Preferences → Media Cache → Clean
  - 프로젝트 저장 후 Premiere Pro 재시작
  - 백그라운드 앱 종료
  - RAM 8GB 미만이면 업그레이드 권장

**문제 7: 클립 미리보기가 느림 (프레임 드롭)**
- **해결**:
  - 프로그램 모니터 → 재생 해상도: 1/4 또는 1/2
  - Sequence → Render In to Out (Enter)
  - GPU 가속 활성화 확인
  - 프록시 워크플로우 사용

#### 2.13 심화 자르기 기법

**Rolling Edit Tool (N 키) 상세**
- 두 클립 사이 경계 조정
- 전체 타임라인 길이 변하지 않음
- **사용법**:
  1. `N` 키로 Rolling Edit Tool 선택
  2. 클립 사이 경계선 드래그
  3. 앞 클립은 짧아지고, 뒤 클립은 길어짐 (또는 반대)
- **용도**: 음악 비트에 맞춰 클립 경계 조정

**Slip Tool (Y 키)**
- 클립의 In/Out 포인트 동시 이동
- 타임라인 위치는 고정
- **사용법**:
  1. `Y` 키로 Slip Tool 선택
  2. 클립 드래그
  3. 클립 내용만 변경 (시작/끝 시점 이동)
- **용도**: 클립 위치는 유지하며 내용만 조정

**Slide Tool (U 키)**
- 클립 자체는 고정, 앞뒤 클립 조정
- **사용법**:
  1. `U` 키로 Slide Tool 선택
  2. 클립 드래그
  3. 앞 클립과 뒤 클립 길이 변경
- **용도**: 전체 타이밍은 유지하며 클립 순서 조정

**도구 비교표**
| 도구 | 단축키 | 전체 길이 변화 | 용도 |
|------|--------|---------------|------|
| Ripple Edit | B | 변함 | 갭 없는 편집 |
| Rolling Edit | N | 변하지 않음 | 클립 경계 조정 |
| Slip Tool | Y | 변하지 않음 | 클립 내용 변경 |
| Slide Tool | U | 변하지 않음 | 클립 타이밍 조정 |

#### 2.14 연습 과제 (실전 훈련)

**기초 과제 1: 5분 영상을 1분으로 압축**
- 목표: 불필요한 부분 제거, 핵심만 남기기
- 시간 제한: 15분
- 평가 기준:
  - [ ] 자연스러운 흐름
  - [ ] 갭 없이 깔끔한 타임라인
  - [ ] 핵심 메시지 전달
- **팁**: 마커로 먼저 킵(keep) 구간 표시

**중급 과제 2: 음악 비트에 맞춰 편집**
- 준비물: 영상 클립 5-10개, 업비트 음악
- 목표: 음악 비트마다 장면 전환
- 단계:
  1. 음악을 타임라인에 먼저 배치
  2. 오디오 파형 확인, 비트 위치에 마커
  3. 마커 위치마다 클립 전환
  4. Q, W 키로 정확한 타이밍 맞추기
- 시간 제한: 20분

**고급 과제 3: B-Roll 인서트 편집**
- 시나리오: 인터뷰 영상 + B-Roll 5개
- 목표: 인터뷰 오디오는 유지, 비주얼만 B-Roll로 교체
- 단계:
  1. 인터뷰를 V1, A1에 배치
  2. B-Roll을 V2에 배치
  3. 인터뷰 영상 필요 없는 부분 숨기기 (눈 아이콘)
  4. B-Roll 타이밍 조정
- 시간 제한: 25분

**속도 훈련 과제: 10-10-10 챌린지**
- 10개 클립을 10분 안에 10초 영상으로 편집
- 규칙:
  - 각 클립은 1초씩 사용
  - 갭 없이 매끄럽게 연결
  - 음악 비트 동기화
- **목표**: 단축키 속도 향상, 근육 기억 형성
