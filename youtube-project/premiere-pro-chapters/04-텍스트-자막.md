### Chapter 4: 텍스트 & 자막
**⭐ 랭킹 쇼츠 재편집 핵심 챕터**

#### 4.1 텍스트/자막 도구 개요

**Premiere Pro의 텍스트 도구**
- **Legacy Title (레거시 타이틀)**: 구버전 도구 (2023년 이후 비권장)
- **Essential Graphics (기본 그래픽)**: 현재 표준 도구 ✅
- **Type Tool (T)**: 타임라인 직접 타이핑

**Essential Graphics 패널 열기**
- Window → Essential Graphics
- 또는 작업 공간을 "Graphics"로 변경

#### 4.2 기본 텍스트 만들기

**방법 1: Type Tool 사용 (가장 빠름)**
1. 툴바에서 Type Tool 선택 (단축키: `T`)
2. 프로그램 모니터 화면 클릭
3. 텍스트 입력
4. Selection Tool (`V`)로 복귀하여 조정

**방법 2: Essential Graphics 패널**
1. Essential Graphics 패널 → "New Layer" → Text
2. 텍스트 입력
3. 패널에서 속성 조정

**텍스트 위치 조정**
- 마우스로 드래그하여 이동
- Effect Controls → Position 값 조정
- 단축키: 화살표 키로 미세 조정

#### 4.3 텍스트 스타일링

**Essential Graphics 패널 주요 속성**

**1. Text (텍스트)**
- **내용 변경**: 텍스트 박스에서 직접 편집
- **폰트**: 시스템 설치 폰트 선택
  - 추천: 나눔고딕, 나눔스퀘어, Pretendard, 맑은고딕
- **크기**: 폰트 사이즈 (쇼츠: 80-120, 롱폼: 60-100)

**2. Appearance (외형)**
- **Fill (채우기)**: 텍스트 색상
  - 흰색 (#FFFFFF), 노란색 (#FFFF00) 많이 사용
- **Stroke (테두리)**: 가독성 향상
  - 두께: 5-15 픽셀
  - 색상: 검정 (#000000) 또는 어두운 색
- **Shadow (그림자)**: 입체감 부여
  - Opacity: 50-80%
  - Distance: 5-10
  - Blur: 10-20

**3. Align and Transform (정렬 및 변형)**
- **Align**: 중앙, 좌측, 우측 정렬
- **Position**: X, Y 좌표
- **Scale**: 크기 조절 (비율 유지)
- **Rotation**: 회전 각도

#### 4.4 자막 스타일 만들기 (쇼츠/유튜브 최적화)

**쇼츠 자막 스타일 (가독성 최우선)**

**설정값 예시**
- **폰트**: Pretendard Bold 또는 나눔스퀘어 Bold
- **크기**: 100-120
- **Fill**: 흰색 (#FFFFFF)
- **Stroke**:
  - 두께: 10-12
  - 색상: 검정 (#000000)
- **배경 박스**: (선택사항)
  - Background → Enable
  - Opacity: 60-80%
  - 색상: 검정

**위치**
- 화면 하단 1/3 지점 (얼굴 가리지 않도록)
- 또는 상단 (콘텐츠에 따라)

**일반 유튜브 자막 스타일**
- **폰트**: 나눔고딕 Regular
- **크기**: 70-90
- **Fill**: 흰색
- **Stroke**: 검정, 두께 6-8
- **위치**: 하단 중앙

#### 4.5 자막 빠르게 복제하기

**단일 자막 복제**
1. 타임라인에서 자막 클립 선택
2. `Alt` + 드래그로 복제
3. 복제된 자막의 텍스트만 변경 (스타일 유지)

**여러 자막 일괄 작업**
1. 첫 자막 완성 (스타일 확정)
2. Essential Graphics → Export → Motion Graphics Template
3. 이후 템플릿 드래그 앤 드롭으로 재사용

#### 4.6 자막 타이밍 조절

**자막 길이 조절**
- 클립 끝부분 드래그
- 권장: 읽는 속도 고려 (2-4초)
- 너무 짧으면 읽기 어려움

**정확한 타이밍 맞추기**
1. 오디오 파형 확인
2. 대사 시작 시점에 자막 시작
3. 대사 끝나는 시점에 자막 종료
4. ←/→ 키로 프레임 단위 조정

**자막 애니메이션 (선택사항)**
- Effect Controls → Opacity 키프레임
  - 시작: 0% → 100% (페이드 인)
  - 종료: 100% → 0% (페이드 아웃)
- 또는 Position 키프레임으로 슬라이드 효과

#### 4.7 모션 그래픽 템플릿 (MOGRT) 활용

**모션 그래픽 템플릿이란?**
- 미리 디자인된 애니메이션 텍스트/그래픽
- After Effects 없이도 전문적인 효과
- .mogrt 파일 형식

**템플릿 찾기**
- **Premiere Pro 내장**: Essential Graphics → Browse
- **무료 사이트**:
  - Mixkit (mixkit.co)
  - Motion Array (일부 무료)
  - Envato Elements (유료)

**템플릿 사용 방법**
1. Essential Graphics 패널 → Browse 탭
2. 카테고리에서 원하는 템플릿 검색
3. 타임라인으로 드래그
4. Edit 탭에서 텍스트/색상 수정

**커스텀 템플릿 설치**
1. .mogrt 파일 다운로드
2. Essential Graphics → 우클릭 → Install Motion Graphics Template
3. 파일 선택 후 설치
4. Local Templates에서 확인

#### 4.8 실전 자막 워크플로우 (쇼츠 편집)

**시나리오**: 30초 쇼츠에 자막 10개 추가

**단계별 작업**

**1. 준비 (1분)**
- 영상 재생하며 자막 필요 구간 파악
- 대략적인 자막 개수 추정

**2. 첫 자막 스타일 설정 (2분)**
- Type Tool (`T`)로 첫 자막 작성
- Essential Graphics에서 스타일 완성
  - 폰트: Pretendard Bold
  - 크기: 110
  - Fill: 흰색, Stroke: 검정 (두께 10)
- 위치: 화면 하단에서 위로 1/4 지점

**3. 자막 복제 및 편집 (5분)**
- Alt + 드래그로 자막 복제
- 각 자막을 대사 타이밍에 맞춰 배치
- 텍스트 내용만 변경 (더블클릭)

**4. 타이밍 정밀 조정 (2분)**
- 재생하며 확인
- 너무 빠르거나 느린 자막 조정
- 클립 길이 드래그로 수정

**총 소요 시간: 약 10분**

#### 4.9 하위 3번째(Lower Third) 자막

**Lower Third란?**
- 화면 하단에 표시되는 이름/직함 자막
- 인터뷰, 설명 영상에서 많이 사용

**만들기**
1. Essential Graphics → Browse → "Lower Third" 검색
2. 템플릿 선택 후 타임라인에 드래그
3. 이름, 직함 텍스트 수정
4. 길이 조절 (3-5초 권장)

**직접 만들기**
1. 사각형 도구로 배경 박스 생성
2. 텍스트 2개 추가 (이름, 직함)
3. 색상/폰트 조정
4. 애니메이션: Position 키프레임으로 슬라이드 인/아웃

#### 4.10 고급 텍스트 효과

**1. 타이핑 효과 (Typewriter)**
- Effect Controls → Text → Source Text 우클릭
- "Separate Characters" 선택
- Opacity 키프레임을 각 글자마다 순차 적용

**2. 텍스트 모션 트래킹**
- 움직이는 대상에 텍스트 따라가게 하기
- Window → Effect Controls
- Effects → Track Motion
- (고급 기능, Chapter 9에서 자세히)

**3. 3D 텍스트 (After Effects 연동)**
- Premiere에서 직접 불가
- After Effects 사용 필요
- 또는 3D 템플릿 활용

#### 4.11 자막 관리 팁

**트랙 분리**
- 자막 전용 비디오 트랙 생성 (V3)
- 메인 영상과 분리하여 관리
- 잠금/숨기기로 편집 효율 증가

**색상 라벨**
- 자막 클립에 특정 색상 지정
- 우클릭 → Label → 색상 선택
- 시각적 구분 용이

**자막 스타일 통일**
- 하나의 영상에서는 일관된 스타일 유지
- 폰트, 크기, 색상 동일하게
- 중요한 부분만 색상 변경 (강조)

#### 4.12 실습 프로젝트 3: 쇼츠 자막 완성하기

**목표**: 이전에 만든 30초 영상에 자막 추가

**단계별 실습**
1. **스타일 설정**
   - 첫 자막을 쇼츠 스타일로 디자인
   - 폰트, 크기, 색상, 테두리 설정

2. **자막 추가**
   - 최소 5개 이상의 자막 배치
   - Alt + 드래그로 복제
   - 내용 및 타이밍 조정

3. **타이틀 추가**
   - 영상 시작 부분에 제목 자막
   - 크기를 크게 (150-200)
   - 1-2초 정도 표시

4. **검토**
   - 전체 재생하며 가독성 확인
   - 자막이 영상 내용과 일치하는지 체크

**체크리스트**
- [ ] 통일된 자막 스타일
- [ ] 5개 이상 자막 배치
- [ ] 타이밍 정확
- [ ] 가독성 확보 (테두리/그림자)
- [ ] 타이틀 자막 추가
- [ ] 프로젝트 저장

**랭킹 쇼츠 특화 팁**
- 순위 숫자는 크고 굵게 (예: **#1**, **#2**)
- 색상으로 순위 구분 (1위: 금색, 2위: 은색, 3위: 동색)
- 설명 텍스트는 상대적으로 작게
- 화면 구성: 순위(좌측) + 설명(우측) + 이미지/영상(중앙)

#### 4.13 자주 하는 실수와 해결법

**실수 1: 자막이 영상에서 잘림 (화면 밖으로 나감)**
- **원인**: Safe Zone 밖에 자막 배치
- **해결**:
  - 프로그램 모니터 → Safe Margins 활성화
  - 자막을 안전 영역 내로 이동
  - 특히 모바일 화면 고려 (여백 확보)
- **예방**: Title Safe 영역 항상 확인 (설정: 10%)

**실수 2: 자막이 너무 작아서 모바일에서 읽기 어려움**
- **원인**: PC 모니터 기준으로 크기 설정
- **해결**:
  - 최소 폰트 크기: 쇼츠 80, 롱폼 60
  - 실제 스마트폰으로 테스트
  - 또는 프로그램 모니터 크기 축소하여 확인
- **기준**: 모니터에서 2m 떨어져서도 읽혀야 함

**실수 3: 텍스트 배경과 구분이 안 됨 (가독성 저하)**
- **원인**: 흰색 배경에 흰색 텍스트, 또는 테두리 없음
- **해결**:
  - Stroke (테두리) 필수 적용: 두께 8-12
  - Shadow (그림자) 추가: Opacity 60-80%
  - 또는 Background 박스 활성화
- **프로 팁**: 배경이 밝으면 어두운 테두리, 어두우면 밝은 테두리

**실수 4: 자막 타이밍이 대사와 안 맞음**
- **원인**: 대충 배치하거나 오디오 파형 미확인
- **해결**:
  - 타임라인 확대 (`+` 키) 후 프레임 단위 조정
  - 오디오 파형 확인하며 정확한 시작/끝 지점 찾기
  - ←/→ 키로 프레임 단위 미세 조정
- **기준**: 대사 시작 0.1초 전, 종료 0.1초 후

**실수 5: 모든 자막 스타일이 다름 (일관성 없음)**
- **원인**: 매번 새로 만들거나 템플릿 미사용
- **해결**:
  - 첫 자막 완성 후 템플릿으로 저장
  - Alt + 드래그로 복제하여 재사용
  - 한 영상에서는 1-2가지 스타일만 사용
- **예외**: 강조할 부분만 색상 변경

**실수 6: 텍스트 애니메이션이 너무 복잡함**
- **원인**: 과도한 효과 남용
- **해결**:
  - 단순한 페이드 인/아웃만으로도 충분
  - 복잡한 애니메이션은 타이틀에만 사용
  - 자막은 즉시 읽혀야 함 (애니메이션 0.3초 이내)
- **원칙**: 내용이 왕, 효과는 보조

**실수 7: 자막 클립이 메인 영상 클립과 섞여서 관리 어려움**
- **원인**: 같은 트랙에 배치
- **해결**:
  - V3 또는 V4 트랙을 자막 전용으로 지정
  - 라벨 색상 적용 (예: 자막은 보라색)
  - 트랙 이름 변경: 우클릭 → Rename → "자막"
- **효과**: 시각적 구분 명확, 편집 효율 향상

#### 4.14 프로 팁 (자막 제작 고급 기법)

**팁 1: 자막 템플릿 라이브러리 구축**
- 자주 쓰는 스타일 10개 정도 템플릿 저장
- 폴더 구조:
  ```
  Essential Graphics/Local Templates/
  ├── 쇼츠_메인자막_화이트
  ├── 쇼츠_강조자막_옐로우
  ├── 롱폼_일반자막
  ├── 타이틀_대형
  └── Lower Third_인터뷰
  ```
- 프로젝트마다 재사용으로 시간 절약 80%

**팁 2: 키프레임으로 동적 자막 만들기**
- **슬라이드 인 효과**:
  1. 자막 클립 선택 → Effect Controls
  2. Position 시작 지점: X = -500 (화면 왼쪽 밖)
  3. 0.3초 후 키프레임: X = 960 (중앙)
  4. Ease In 적용 (부드러운 감속)
- **줌 인 효과**:
  1. Scale 시작: 50%
  2. 0.2초 후: 100%
  3. Ease Out 적용
- **용도**: 타이틀, 중요 공지사항

**팁 3: 자막 정렬 단축키 활용**
- 텍스트 선택 후 Essential Graphics → Align:
  - 수평 중앙: Horizontal Center
  - 수직 중앙: Vertical Center
  - 또는 Effect Controls → Anchor Point 조정
- **빠른 방법**: 텍스트 드래그 중 Shift 누르면 직선 이동

**팁 4: 반복 자막 일괄 변경**
- **시나리오**: "구독" 자막 10개를 "좋아요"로 변경
- **방법**:
  1. Find → Text 탭
  2. "구독" 검색 → 모든 인스턴스 표시
  3. Replace All → "좋아요"
  4. 한 번에 모두 변경
- **주의**: Premiere Pro 일부 버전에서만 지원

**팁 5: 이중 자막 (한국어 + 영어) 효율 배치**
- **레이아웃**:
  - V3: 한국어 자막 (하단)
  - V4: 영어 자막 (상단 또는 한국어 아래)
- **스타일 구분**:
  - 한국어: 큰 폰트 (100), 흰색
  - 영어: 작은 폰트 (60), 회색
- **동기화**: 두 자막 클립을 Nest하여 묶기

**팁 6: 자막 색상 빠른 변경 (브랜드 색상 적용)**
- Essential Graphics → Master Style 사용
- 색상 팔레트 미리 설정:
  - 브랜드 색상 1: #FF5733
  - 브랜드 색상 2: #33FF57
  - 중립 색상: #FFFFFF, #000000
- 클릭 한 번으로 일괄 변경

**팁 7: 긴 자막 자동 줄바꿈 설정**
- Essential Graphics → Text → Paragraph
- Word Wrap 활성화
- Max Width 설정 (예: 1600px)
- 자동으로 적절히 줄바꿈
- **주의**: 수동 줄바꿈이 더 자연스러울 때 많음

#### 4.15 실전 예시: 랭킹 쇼츠 자막 시스템

**시나리오**: Top 5 랭킹 쇼츠 자막 완전 가이드

**자막 레이어 구조**
- **V5**: 순위 숫자 (#1, #2, #3...)
- **V4**: 타이틀/제목
- **V3**: 설명 텍스트
- **V2**: 영상 클립
- **V1**: 배경

**순위 숫자 스타일 (V5)**
```
폰트: Impact 또는 Pretendard ExtraBold
크기: 180
위치: 좌측 상단 (X: 200, Y: 300)
색상:
  - 1위: 금색 그라데이션 (#FFD700 → #FFA500)
  - 2위: 은색 (#C0C0C0)
  - 3위: 동색 (#CD7F32)
  - 4-5위: 흰색 + 두꺼운 검정 테두리
Stroke: 15, 검정
Shadow: Distance 10, Blur 15, Opacity 80%
```

**타이틀 스타일 (V4)**
```
폰트: Pretendard Bold
크기: 90
위치: 중앙 상단 (Y: 250)
색상: 흰색
Stroke: 10, 검정
배경 박스: 검정, Opacity 70%, Padding 20
```

**설명 텍스트 (V3)**
```
폰트: Pretendard Regular
크기: 65
위치: 하단 (Y: 1600)
색상: 흰색
Stroke: 8, 검정
최대 2줄, 줄간격 1.2
```

**애니메이션 타이밍**
- 순위 숫자: 슬라이드 인 (좌측에서, 0.3초)
- 타이틀: 페이드 인 + 작은 줌 (0.2초)
- 설명: 순차 페이드 인 (0.15초 간격)
- 클립 전환 시: 모든 자막 페이드 아웃 동시 (0.2초)

**제작 시간**: 5개 항목 자막 완성 15분

#### 4.16 텍스트 효과 고급 기법

**1. 텍스트 뒤 블러 배경 만들기**
- 목적: 복잡한 배경에서 가독성 향상
- 방법:
  1. 조정 레이어 생성 (V2)
  2. Gaussian Blur 이펙트 적용 (50-100)
  3. 마스크로 텍스트 영역만 선택
  4. Feather: 100 (부드러운 경계)
  5. 텍스트 레이어 (V3) 위에 배치
- **결과**: 텍스트 뒤만 블러 처리

**2. 그라데이션 텍스트**
- Premiere Pro 자체 기능 제한적
- **해결책**:
  - After Effects에서 제작 후 MOGRT로 저장
  - 또는 4-Color Gradient 이펙트 + 마스크 조합
  - 또는 그라데이션 이미지를 Track Matte로 사용
- **간단한 방법**: 템플릿 사이트에서 다운로드

**3. 네온 사인 효과**
- 텍스트에 Glow 이펙트 적용
- 설정:
  - Glow Threshold: 50%
  - Glow Radius: 30-50
  - Glow Intensity: 1.5-2.0
  - 색상: 밝은 네온 색 (핑크, 시안, 라임)
- **추가**: Gaussian Blur (5-10) 중첩

**4. 글리치(Glitch) 텍스트**
- VR Digital Glitch 이펙트 사용
- 또는 Wave Warp + RGB Split 조합
- 키프레임으로 순간적으로만 적용 (0.1초)
- **용도**: 게임, 테크, 트렌디 콘텐츠

#### 4.17 문제 해결 (자막 관련)

**문제 1: 텍스트를 입력했는데 화면에 안 보임**
- **원인**: 텍스트 색상이 배경과 동일하거나 크기 0
- **해결**:
  - Effect Controls → Fill → 색상 변경
  - Font Size 확인 (최소 60 이상)
  - Opacity 100% 확인
- **디버깅**: 텍스트 선택 → 바운딩 박스 확인

**문제 2: 자막 클립이 너무 길거나 짧게 표시됨**
- **원인**: 클립 길이와 표시 시간 혼동
- **해결**:
  - 타임라인에서 자막 클립 끝 드래그로 길이 조정
  - 또는 속도 조절 (우클릭 → Speed/Duration)
- **기본 길이**: 새 텍스트 클립은 5초 기본값

**문제 3: 템플릿이 설치되지 않음**
- **원인**: 파일 경로 오류 또는 권한 문제
- **해결**:
  - .mogrt 파일을 직접 드래그 앤 드롭 (타임라인)
  - 또는 Local Templates 폴더로 수동 이동
  - 폴더 위치: Documents/Adobe/Premiere Pro/[버전]/Motion Graphics Templates
- **재시작**: Premiere Pro 재시작 후 확인

**문제 4: 자막 편집 중 프로그램이 느려짐**
- **원인**: 복잡한 텍스트 효과, 많은 레이어
- **해결**:
  - 자막 클립을 네스트 시퀀스로 그룹화
  - 또는 자막 완성 후 Render and Replace
  - GPU 가속 확인
- **임시 해결**: 편집 중 Shadow, Glow 비활성화

**문제 5: 폰트가 목록에 없음**
- **원인**: 시스템에 폰트 미설치
- **해결**:
  - 폰트 파일 다운로드 (.ttf 또는 .otf)
  - Windows: C:\Windows\Fonts에 복사
  - Mac: Font Book으로 설치
  - Premiere Pro 재시작
- **주의**: 상업용 폰트 라이선스 확인 필수

**문제 6: 자막 복제 시 위치가 틀어짐**
- **원인**: Anchor Point 중심점 설정 다름
- **해결**:
  - Effect Controls → Anchor Point 값 확인
  - 또는 Position 값을 수동으로 동일하게 설정
  - 템플릿 사용 시 이 문제 없음
- **예방**: 첫 자막 만들 때 중심점 확정

#### 4.18 연습 과제 (자막 마스터)

**기초 과제 1: 10초 영상에 자막 5개 추가**
- 목표: 기본 자막 제작 및 타이밍
- 요구사항:
  - [ ] 통일된 스타일 (폰트, 크기, 색상)
  - [ ] 정확한 타이밍 (대사 동기화)
  - [ ] 테두리 또는 그림자로 가독성 확보
  - [ ] 화면 안전 영역 내 배치
- 시간 제한: 10분

**중급 과제 2: Lower Third 3가지 스타일 제작**
- 목표: 템플릿 제작 능력 향상
- 스타일:
  1. 미니멀 (텍스트 + 얇은 라인)
  2. 박스형 (배경 박스 + 텍스트)
  3. 애니메이션 (슬라이드 인/아웃)
- 각 템플릿 저장 및 재사용 테스트
- 시간 제한: 30분

**고급 과제 3: 랭킹 쇼츠 자막 시스템 구축**
- 목표: 완전한 자막 워크플로우 완성
- 요구사항:
  - Top 3 순위 자막 (금, 은, 동 색상)
  - 각 순위마다 타이틀 + 설명 텍스트
  - 애니메이션 (슬라이드/줌/페이드)
  - 일관된 타이밍 (각 3-5초)
- 최종 산출물: 15초 완성 영상
- 시간 제한: 40분

**창의성 과제: 브랜드 자막 스타일 개발**
- 목표: 독창적인 자막 스타일 창조
- 단계:
  1. 브랜드 색상 2-3개 선정
  2. 폰트 조합 (타이틀용, 본문용)
  3. 고유한 레이아웃 디자인
  4. 애니메이션 시그니처 개발
  5. 템플릿으로 저장
- **보너스**: 5개 영상에 동일 스타일 적용하여 일관성 확인
