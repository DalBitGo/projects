### Chapter 7: 쇼츠 영상 제작

#### 7.1 쇼츠(Shorts) 포맷 이해

**쇼츠 플랫폼**
- YouTube Shorts
- TikTok
- Instagram Reels
- 모두 세로 형식 (9:16 비율)

**쇼츠 성공 요소**
- 첫 3초가 승부 (시선 잡기)
- 빠른 편집 리듬
- 큰 자막 (가독성)
- 트렌디한 BGM
- 명확한 메시지

#### 7.2 9:16 세로 시퀀스 설정

**방법 1: 직접 설정**
1. File → New → Sequence
2. Settings 탭 클릭
3. Frame Size:
   - **Horizontal**: 1080
   - **Vertical**: 1920
4. Frame Rate: 30fps 또는 60fps
5. Sequence Name: "Shorts_01"

**방법 2: 프리셋 사용**
1. ARRI → 1080p 선택
2. Settings에서 Frame Size 확인
3. 이미 9:16 비율로 설정됨

**가로 영상을 세로로 변환**
1. 가로 영상 클립 선택
2. Effect Controls → Motion → Scale: 150-200%
3. Position 조정하여 중요한 부분 중앙 배치

#### 7.3 빠른 컷 편집 기법

**Jump Cut (점프 컷)**
- 같은 장면에서 불필요한 부분 잘라내기
- 리듬감 있게 편집
- 0.5-2초 간격으로 빠르게 컷

**워크플로우**
1. 전체 클립 배치
2. 말 더듬거나 침묵 구간 삭제
3. `Ctrl + K`로 분할 → Delete
4. Ripple Delete로 갭 제거
5. 빠른 템포 유지

**Beat Sync 편집**
- BGM 비트에 맞춰 컷
- M 키로 비트마다 마커 표시
- 마커 위치에서 클립 전환

#### 7.4 트렌디한 쇼츠 이펙트

**1. 줌인/줌아웃 (Punch In/Out)**
- 강조하고 싶은 순간에 사용
- Effect Controls → Scale
- 키프레임: 100% → 110% (0.2초)
- 빠르게 확대했다가 복구

**2. 속도 조절 (Speed Ramp)**
- 일반 속도 → 슬로우 → 일반 속도
- 클립 우클릭 → Speed/Duration
- 또는 Time Remapping 사용:
  1. 클립 우클릭 → Show Clip Keyframes → Time Remapping → Speed
  2. Ctrl + 클릭으로 키프레임 추가
  3. 드래그하여 속도 조절

**3. Glitch 효과**
- Effects → Video Effects → Distort → Wave Warp
- Amount: 5-10
- 짧게 (0.1초) 적용하여 글리치 느낌

**4. VHS/Retro 효과**
- Effects → Sharpen → Unsharp Mask (선명도 낮춤)
- Lumetri Color → Faded Film: 50
- 노이즈 추가 (Noise & Grain)

**5. 화면 흔들림 (Camera Shake)**
- Effect Controls → Position
- 키프레임 여러 개 빠르게 추가
- 랜덤한 값으로 흔들림 연출

#### 7.5 자막 최적화 (쇼츠 특화)

**쇼츠 자막 원칙**
- 큰 폰트 (100-140)
- 굵은 테두리 (10-15)
- 화면 중앙 또는 하단 1/3
- 1-2줄 최대 (너무 길면 읽기 어려움)
- 단어별로 강조 (색상 변경)

**애니메이션 자막**
- Position 키프레임: 아래→위 슬라이드
- Opacity 키프레임: 페이드 인/아웃
- Scale 키프레임: 팝업 효과

#### 7.6 쇼츠 편집 워크플로우

**30초 쇼츠 제작 (전체 30분)**

**1. 기획 (5분)**
- 주제 정하기
- 스크립트 작성 (간단히)
- 후크(Hook) 정하기 (첫 3초 멘트)

**2. 촬영 (10분)**
- 세로 모드로 촬영
- 여러 테이크 찍기
- 밝은 곳에서 촬영

**3. 편집 (10분)**
- 9:16 시퀀스 생성
- 클립 배치 및 빠른 컷
- 불필요한 부분 제거
- 30초 이내로 조정

**4. 자막 & 효과 (3분)**
- 주요 멘트에 자막 추가
- 1-2개 줌인 효과

**5. BGM & 사운드 (2분)**
- 트렌디한 BGM 추가
- 덕킹 적용

**6. 내보내기 (시간은 컴퓨터 성능에 따라)**
- H.264, 1080x1920, 30fps

#### 7.8 자주 하는 실수와 해결법

**실수 1: 자막이 화면 상단/하단에 잘림**
- **원인**: 모바일 UI (좋아요 버튼, 댓글 영역)를 고려하지 않음
- **증상**: 업로드 후 자막이 UI에 가려짐
- **해결**:
  - 자막 안전 영역 준수: 상단 15%, 하단 25% 비우기
  - Essential Graphics → Safe Margins 표시 (View → Safe Margins)
  - 자막 위치를 화면 중앙~중상단에 배치
- **권장 위치**: Y 포지션 400-700 (1920 기준)
- **테스트**: 실제 스마트폰에서 확인

**실수 2: 세로 영상인데 옆에 검은 여백이 생김**
- **원인**: 원본 영상이 가로(16:9)인데 Scale만 조정
- **해결**:
  - 방법 1: Scale을 200-250%로 확대하여 화면 채우기
  - 방법 2: Crop 이펙트로 좌우 잘라내기
  - 방법 3: 애초에 세로로 촬영 (권장)
- **Crop 설정**:
  - Effects → Transform → Crop
  - Left: 20%, Right: 20%
  - Scale to Frame Size 체크

**실수 3: 첫 3초에 후킹 포인트 없어 이탈률 높음**
- **원인**: 느리게 시작하거나 인트로가 김
- **증상**: 조회수는 있는데 평균 시청 시간 5초 미만
- **해결**:
  - 첫 1-3초에 핵심 메시지 또는 충격적인 장면 배치
  - "이거 모르면 후회함", "믿을 수 없는 결과" 같은 후킹 멘트
  - 인트로 로고는 생략 또는 2초 이내
- **예시**: "오늘 30만원 벌었습니다" (첫 문장부터 임팩트)
- **데이터 확인**: YouTube Analytics → Audience Retention 그래프

**실수 4: 음악 비트와 컷이 안 맞음**
- **원인**: 영상을 먼저 자르고 BGM을 나중에 추가
- **해결**:
  - BGM을 먼저 타임라인에 배치
  - 오디오 파형 보며 비트 위치에 마커 추가 (M 키)
  - 마커에 맞춰 영상 컷 편집
  - 또는 Rate Stretch Tool (X)로 클립 길이를 비트에 맞춤
- **프로 기준**: 비트와 컷이 일치하면 전문성 2배 증가

**실수 5: 자막 폰트가 너무 작거나 가독성 낮음**
- **원인**: 데스크톱 모니터에서만 확인, 모바일 고려 안 함
- **해결**:
  - 최소 폰트 크기: 80pt 이상 (1920 높이 기준)
  - 고대비 색상: 흰색 텍스트 + 검은색 외곽선 (4-6pt)
  - 또는 텍스트 박스 배경 (반투명 검정 사각형)
- **테스트**: 스마트폰에서 5미터 떨어져서 읽히는지 확인
- **폰트 추천**: Pretendard Bold, 나눔스퀘어 ExtraBold

**실수 6: 쇼츠 길이가 60초 초과**
- **원인**: 타임라인 시간 확인 안 함
- **증상**: YouTube Shorts로 인식 안 됨, 일반 영상으로 업로드
- **해결**:
  - 타임라인 하단 시간 표시 확인 (00:00:00 ~ 00:00:59)
  - 안전하게 55-58초로 마무리
  - 여유분 2-5초 남겨두기
- **최적 길이**:
  - 짧은 쇼츠: 15-30초 (재시청률 높음)
  - 표준 쇼츠: 40-55초

**실수 7: 과도한 이펙트와 트랜지션으로 정신없음**
- **원인**: 이펙트를 재미있다고 생각하고 남발
- **증상**: 시청자 피로도 증가, 메시지 전달 방해
- **해결**:
  - 이펙트는 강조할 때만 사용 (핵심 장면 2-3곳)
  - 트랜지션은 Cross Dissolve로 통일
  - 줌인/줌아웃은 1-2회만
- **프로 기준**: "Less is More" - 심플할수록 전문적

#### 7.9 프로 팁

**팁 1: 템플릿화로 제작 속도 10배 향상**
- 자주 만드는 쇼츠 스타일을 템플릿으로 저장
- **방법**:
  1. 완성된 쇼츠 시퀀스 복제
  2. 클립만 삭제, 자막/이펙트/BGM 구조는 유지
  3. 이름: "Shorts_Template_TopList"
  4. 다음 영상은 새 클립만 교체
- **템플릿 예시**:
  - Top 3 리스트형
  - Before/After 비교형
  - 튜토리얼형 (단계별 자막)
- **효과**: 첫 제작 30분 → 템플릿 사용 시 5-10분

**팁 2: Auto Reframe으로 AI 자동 크롭**
- Premiere Pro AI가 자동으로 세로 영상 생성
- **사용법**:
  1. 가로 영상 선택
  2. Sequence → Auto Reframe Sequence
  3. Target Aspect Ratio: Vertical (9:16)
  4. Motion Tracking: Default (또는 Slower Motion)
  5. Create Sequence
- **장점**: AI가 피사체 따라가며 자동 크롭
- **단점**: 100% 완벽하진 않음, 수동 보정 필요 (20%)
- **활용**: 기존 가로 영상을 쇼츠로 재활용

**팁 3: 자막 애니메이션 프리셋 구축**
- 매번 자막 애니메이션 만들지 말고 프리셋 활용
- **인기 애니메이션**:
  - 페이드 인 + 스케일 (100% → 110% → 100%)
  - 슬라이드 인 (아래→위)
  - 타이핑 효과 (한 글자씩 등장)
- **프리셋 저장**:
  1. 자막에 애니메이션 적용
  2. Effect Controls → 우클릭 → Save Preset
  3. 이름: "Shorts_Text_PopIn"
- **적용**: 다른 자막에 드래그 앤 드롭

**팁 4: 음악 비트 마커를 타임라인에 자동 배치**
- 수동으로 비트 찾지 말고 AI 활용
- **방법**:
  1. BGM 클립 우클릭 → Edit Clip in Adobe Audition
  2. Audition에서 Window → Markers
  3. Remix → Beat Detection
  4. Sensitivity 조정 → OK
  5. 비트마다 자동으로 마커 생성
  6. Save → Premiere에 자동 반영
- **또는**: 온라인 툴 "BeatEdit" (유료 플러그인)
- **활용**: 마커 위치에 정확히 컷 편집

**팁 5: 쇼츠 전용 색보정 프리셋 (모바일 최적화)**
- 모바일은 색상이 더 밝고 채도가 높게 보임
- **모바일 최적화 설정**:
  - Contrast: +25 (강한 대비)
  - Saturation: +15 (선명한 색)
  - Highlights: -15 (밝은 부분 보호)
  - Sharpness: +20 (선명도)
- **프리셋 저장**: "Shorts_Mobile_Optimized"
- **테스트**: 실제 스마트폰에서 확인 필수

**팁 6: 썸네일 프레임을 동영상 안에 숨기기**
- 쇼츠는 썸네일을 따로 업로드 못함, 영상 프레임이 썸네일
- **전략**:
  - 가장 임팩트 있는 프레임을 영상 시작(0-3초) 또는 중간(15-20초)에 배치
  - 해당 프레임에 큰 텍스트 오버레이 ("충격", "대박" 등)
  - 그래픽 요소 추가 (화살표, 원, 강조선)
- **A/B 테스트**: 여러 버전 업로드 후 클릭률 비교

**팁 7: 쇼츠 시리즈로 연속 시청 유도**
- 하나의 주제를 여러 편으로 나눠 시리즈 제작
- **예시**:
  - "Premiere Pro 꿀팁 #1, #2, #3..."
  - "30일 챌린지 Day 1, Day 2..."
- **장점**:
  - 구독자 유입 증가
  - 시청자가 다음 편 찾아봄
  - 알고리즘이 연관 쇼츠 추천
- **제목 패턴**: "[주제] #시리즈번호 - [부제목]"

#### 7.10 실전 예시: 바이럴 쇼츠 제작 A-Z

**시나리오**: "내가 1달 동안 Premiere Pro 배워서 만든 결과" (30초 Before/After 쇼츠)

**기획 단계 (5분)**
1. 컨셉 정하기: Before(서툰 편집) vs After(프로 편집) 비교
2. 스토리보드 스케치:
   - 0-3초: 후킹 ("1달 전 vs 지금")
   - 3-15초: Before 영상 (어설픈 편집 사례)
   - 15-27초: After 영상 (프로 편집)
   - 27-30초: CTA ("구독하면 당신도 가능")
3. 필요 소스:
   - Before 클립: 5-7개
   - After 클립: 5-7개
   - BGM: 업비트, 30초
   - 자막 텍스트 준비

**1단계: 시퀀스 및 소스 준비 (3분)**
1. 새 시퀀스: 1080x1920, 30fps, 이름 "Shorts_BeforeAfter_v01"
2. 소스 임포트:
   - Before 영상 (의도적으로 조잡하게 편집된 클립)
   - After 영상 (깔끔한 편집)
   - BGM (Epidemic Sound 또는 YouTube Audio Library)
3. 프로젝트 패널에서 폴더 정리

**2단계: 타이밍 구조 설정 (5분)**
1. BGM을 A1 트랙에 먼저 배치
2. 30초 길이에 맞춰 BGM 자르기
3. 비트 위치에 마커 추가 (M 키):
   - 0초: 시작
   - 3초: Before 시작
   - 15초: After 시작
   - 27초: CTA 시작
   - 30초: 끝
4. 마커 사이를 클립으로 채울 계획

**3단계: Before 섹션 편집 (7분)**
1. 3-15초 구간에 Before 클립 5개 배치
2. 각 클립 길이: 2-3초 (빠른 리듬)
3. 일부러 조잡한 느낌 연출:
   - 자막 없음 또는 작은 글씨
   - 컷이 어색하게 연결
   - 색보정 없음
4. "1달 전" 텍스트 오버레이:
   - 폰트: Arial, 60pt (작게)
   - 위치: 화면 구석
   - 색상: 흰색 (외곽선 없음, 가독성 낮게)

**4단계: After 섹션 편집 (10분)**
1. 15-27초 구간에 After 클립 5개 배치
2. 각 클립 길이: 2-3초, 비트에 정확히 맞춤
3. 프로 편집 요소 적용:
   - 큰 자막 (120pt, 굵은 글씨, 외곽선 6pt)
   - 색보정 (Teal & Orange Look, Contrast +20)
   - 부드러운 트랜지션 (Cross Dissolve 0.5초)
   - 줌인 효과 (Scale 100% → 110%, 1초)
4. "지금" 텍스트 오버레이:
   - 폰트: Pretendard ExtraBold, 140pt
   - 위치: 화면 중상단
   - 색상: 노란색, 검은 외곽선 6pt, 드롭 섀도우

**5단계: 후킹 인트로 제작 (5분)**
1. 0-3초 구간:
   - 분할 화면 (Split Screen)
   - 왼쪽: Before 대표 장면
   - 오른쪽: After 대표 장면
2. Effects → Crop으로 각 클립 반씩 자르기:
   - Before: Right 50% 크롭
   - After: Left 50% 크롭
3. 중앙에 흰색 구분선 (Rectangle 그래픽)
4. 상단 텍스트: "1달 전 VS 지금"
   - 폰트: 180pt, 굵게
   - 애니메이션: 아래에서 슬라이드 인 (0.5초)

**6단계: CTA (Call To Action) 추가 (3분)**

**8단계: 최종 색보정 및 샤프닝 (5분)**

#### 7.11 쇼츠 고급 테크닉

**테크닉 1: 다중 화면 레이아웃 (Split Screen)**
- 하나의 화면에 2-4개 영상 동시 표시
- **2분할 (상하 또는 좌우)**:
  1. 두 클립을 V1, V2에 배치
  2. V2 클립 선택 → Effect Controls
  3. Position: Y 값 변경 (상단: -480, 하단: +480)
  4. Scale: 100% 유지
  5. 또는 Crop으로 반씩 자르기
- **4분할 (Grid)**:
  1. 4개 클립을 V1-V4에 배치
  2. 각 Position 설정:
     - 좌상: X -270, Y -480
     - 우상: X +270, Y -480
     - 좌하: X -270, Y +480
     - 우하: X +270, Y +480
  3. Scale: 50%
- **활용**: 비교 영상, 반응 영상, 멀티캠

**테크닉 2: 동적 자막 (Kinetic Typography)**
- 자막이 움직이며 강조
- **바운스 효과**:
  1. 텍스트 레이어 선택
  2. Effect Controls → Scale
  3. 키프레임:
     - 0초: 0% (시작)
     - 0.3초: 120% (오버슈트)
     - 0.5초: 100% (정착)
- **흔들림 효과**:
  1. Position에 Wiggle Expression (After Effects 스타일)
  2. 또는 수동 키프레임:
     - 좌: -10px
     - 우: +10px
     - 반복 (0.1초 간격)
- **활용**: 핵심 단어 강조, 충격적인 멘트

**테크닉 3: 역재생 효과**
- 시간을 거꾸로 돌리는 효과 (리와인드)
- **방법**:
  1. 클립 우클릭 → Speed/Duration
  2. Reverse Speed 체크
  3. OK
  4. 또는 Rate Stretch Tool (X)로 클릭
- **창의적 활용**:
  - 떨어진 물건이 다시 올라옴
  - 파괴된 것이 복원됨
  - 음식 먹방 역재생 (토해내는 효과)

**테크닉 4: 스톱 모션 효과**
- 부드러운 영상을 끊어지는 느낌으로
- **Posterize Time 이펙트**:
  1. Effects → Time → Posterize Time
  2. Frame Rate: 12fps (영화적) 또는 8fps (더 끊김)
- **수동 방법**:
  1. 클립의 프레임을 5프레임마다 하나씩만 유지
  2. 나머지 프레임 삭제
- **활용**: 역동적인 느낌, 뮤직비디오 스타일

**테크닉 5: 글리치 효과 (Glitch Effect)**
- 화면이 깨지고 왜곡되는 효과
- **VHS 글리치**:
  1. Effects → Distort → Wave Warp
  2. Wave Height: 10-20
  3. Wave Width: 100-200
  4. 키프레임으로 짧게 (0.1초) 깜빡임
- **RGB Split**:
  1. Effects → Channel → Channel Blur
  2. Red/Green/Blue 각각 다른 방향 이동
  3. Red: +5px, Green: -3px, Blue: +7px
- **활용**: 전환 효과, 충격 장면

**테크닉 6: 타임랩스를 쇼츠로**
- 긴 타임랩스를 쇼츠에 최적화
- **Speed Ramping**:
  1. 타임랩스 클립 배치 (예: 5분 분량)
  2. 핵심 구간만 추출 (10-20초)
  3. Speed/Duration → 200-400% (더 빠르게)
  4. 핵심 순간만 100% (정상 속도)로 강조
- **Time Remapping**:
  1. 클립 우클릭 → Show Clip Keyframes → Time Remapping → Speed
  2. 키프레임으로 속도 변화 (빠름 → 느림 → 빠름)
- **활용**: 일출/일몰, 요리 과정, 작업 과정

**테크닉 7: Green Screen (크로마키) 합성**
- 배경 교체로 창의적 영상 제작
- **워크플로우**:
  1. 초록 배경에서 촬영한 영상 준비
  2. V1에 새 배경 영상/이미지 배치
  3. V2에 초록 배경 인물 영상 배치
  4. Effects → Keying → Ultra Key 적용 (V2)
  5. Eyedropper로 초록색 클릭
  6. Key Color Tolerance/Pedestal 조정 (깔끔한 키잉)
  7. Spill Suppression으로 초록 반사 제거
- **쇼츠 활용**:
  - 판타지 배경 (우주, 언더워터)
  - 뉴스 앵커 스타일
  - 반응 영상 (내가 영화 속에)

#### 7.12 문제 해결

**문제 1: 쇼츠가 YouTube Shorts 피드에 안 뜸**
- **원인**: 세로 비율이 아니거나, 길이 초과, 또는 "#Shorts" 태그 누락
- **해결**:
  - 해상도 확인: 정확히 1080x1920 (9:16)
  - 길이 확인: 60초 이하 (안전하게 58초)
  - 제목 또는 설명에 "#Shorts" 또는 "#shorts" 추가
  - 업로드 48시간 대기 (알고리즘 인식 시간)

**문제 2: 모바일에서 자막이 UI에 가려짐**
- **원인**: Safe Margin 무시
- **해결**:
  - 자막 Y Position 범위: 400-700 (1920 기준)
  - 하단 25% (1440-1920), 상단 15% (0-288) 피하기
  - Essential Graphics → View → Safe Margins 켜고 작업
  - 실기기 테스트 필수

**문제 3: 내보낸 영상이 옆으로 누워있음 (회전됨)**
- **원인**: 스마트폰 세로 촬영 시 메타데이터 회전 정보
- **해결**:
  - 클립 우클릭 → Modify → Interpret Footage
  - Rotation: 90도 또는 270도 선택
  - OK → 자동 회전
  - 또는 Effect Controls → Rotation에서 수동 회전

**문제 4: 음악 저작권 문제로 영상 차단**
- **원인**: 저작권 있는 BGM 사용
- **해결**:
  - YouTube Audio Library 사용 (무료, 저작권 안전)
  - Epidemic Sound, Artlist 구독 (유료, 상업 사용 가능)
  - 저작권 Free 음악만 사용
  - 또는 자작 음악
- **확인 방법**: YouTube Studio → 저작권 확인

**문제 5: 쇼츠 조회수는 많은데 구독자 증가 적음**
- **원인**: CTA (Call To Action) 부족
- **해결**:
  - 영상 마지막 3-5초에 구독 유도 멘트
  - "더 보려면 구독!", "다음 편도 기대하세요"
  - 손가락 포인팅 그래픽 추가
  - 시리즈물로 연속성 제공

**문제 6: 색보정이 모바일에서 이상하게 보임**
- **원인**: 데스크톱 모니터와 모바일 디스플레이 색감 차이
- **해결**:
  - 모바일 실기기에서 항상 최종 확인
  - Saturation 너무 높이지 않기 (+15 이내)
  - Contrast 적당히 (+20-25)
  - 다양한 밝기 환경에서 테스트

**문제 7: 쇼츠 업로드 후 화질 저하**
- **원인**: YouTube 압축
- **해결**:
  - 내보내기 시 Bitrate 높게: 15-20 Mbps VBR
  - Resolution: 정확히 1080x1920 (업스케일/다운스케일 피하기)
  - Export Settings → Use Maximum Render Quality 체크
  - H.264 코덱, High Profile, Level 4.2
