# NotebookLM Video Overview 사용 가이드

## 1. NotebookLM 시작하기

### 접속
- URL: https://notebooklm.google/
- 개인 Google 계정으로 로그인 (만 18세 이상)
- 데스크톱 브라우저 사용 (모바일 앱은 Video Overview 미지원)

### 노트북 생성
1. "New Notebook" 클릭
2. 주제별로 노트북 분리 (한 노트북 = 한 주제)

## 2. 자료 업로드

### 지원 형식
- **문서**: PDF, Google Docs, Google Slides
- **웹**: URL (기사, 블로그 등)
- **영상**: YouTube 동영상
- **오디오**: MP3, WAV 등
- **텍스트**: 직접 복사/붙여넣기

### 업로드 팁
✅ **표·다이어그램·이미지를 문서에 포함**
   - NotebookLM이 자동으로 슬라이드에 반영
   - 숫자, 인용구도 자동 추출

✅ **구조화된 문서 준비**
   - 명확한 제목/부제목
   - 핵심 포인트 강조
   - 이미지에 캡션 추가

❌ **피해야 할 것**
   - 여러 주제 섞어서 업로드
   - 저작권 확인 안 된 자료
   - 저해상도 이미지

## 3. Video Overview 생성

### 기본 단계
1. Studio 탭 클릭
2. "Video Overview" 선택
3. 설정 커스터마이즈
4. "Generate" 클릭
5. 생성 완료 (수 분 소요)

### 커스터마이징 옵션

#### 포맷 (Format)
- **Explainer**: 상세 설명 (3-5분)
- **Brief**: 짧은 요약 (1-2분, 쇼츠용)

#### 대상 (Audience)
- 초급자 (Beginner)
- 일반인 (General)
- 전문가 (Expert)

#### 스타일 (Visual Style)
- **Whiteboard**: 화이트보드 스타일
- **Watercolor**: 수채화 느낌
- **Retro Print**: 레트로 프린트
- 기타 다양한 스타일

#### 언어 (Output Language)
- Settings → Output Language
- 80+ 언어 지원 (한국어 포함)

### 프롬프트 커스터마이징 예시

```
"이 표 중심으로 핵심 수치만 설명해. 증감률은 소수점 1자리까지만."
```

```
"초보자 대상으로, 90초 이내 요약. 유튜브 쇼츠용."
```

```
"전문 용어 사용 가능. 데이터 분석 중심으로 설명."
```

```
"유튜브 인트로용 후킹 문장 1개 추가. 핵심만 60초 요약."
```

## 4. 품질 체크리스트

### 생성 후 확인사항
- [ ] 전체 재생해서 내레이션 잘림 확인
- [ ] 표/숫자가 원본과 일치하는지 확인
- [ ] 장면 전환이 자연스러운지 확인
- [ ] 발음/억양이 적절한지 확인

### 문제 발생 시
- **내레이션 잘림**: 다시 생성 또는 Brief → Explainer로 변경
- **숫자 오류**: 원본 문서에서 숫자 명확히 표시
- **어색한 흐름**: 문서 구조 개선 후 재생성

## 5. 다운로드 & 공유

### MP4 다운로드
- 생성 완료 후 "Download" 버튼
- 16:9 비율 (YouTube 표준)
- 고화질 MP4

### 공유 옵션
- 노트북 링크 공유 (View 권한 설정)
- 직접 다운로드 후 업로드

## 6. YouTube 업로드 최적화

### 업로드 전 체크
- [ ] 해상도: 1080p 권장
- [ ] 비율: 16:9 (일반) 또는 9:16 (쇼츠)
- [ ] 자막: 한국어 자동자막 + 수동 교정
- [ ] 썸네일: 핵심 키워드/숫자 강조
- [ ] 설명란: 출처 명시

### 메타데이터 작성
```
제목: [핵심 키워드] + [숫자/결과] + [후킹 요소]
예: "2024 부동산 시장 -32.7% 하락, 5가지 핵심 분석"

설명:
- 영상 출처: NotebookLM 자동 생성
- 데이터 출처: [원본 문서/링크]
- 타임스탬프 추가
```

## 7. 주의사항

### 언어 지원
- 현재 80+ 언어 지원
- 한국어 품질은 계속 개선 중
- 영어가 가장 안정적

### 데스크톱 전용
- 모바일 앱에서는 Video Overview 생성 불가
- 재생/공유는 모바일에서도 가능

### 저작권
- 업로드한 자료의 저작권 확인 필수
- 공개 자료/자체 제작 자료 권장
- 회사 내부 자료는 보안 정책 확인

### AI 생성 컨텐츠
- YouTube 업로드 시 "AI 생성 콘텐츠" 체크 권장
- 사실 확인은 반드시 수동으로

## 8. 트러블슈팅

| 문제 | 해결 방법 |
|------|----------|
| 생성 실패 | 문서 크기 줄이기, 이미지 최적화 |
| 영어로 생성됨 | Settings → Output Language 확인 |
| 품질 낮음 | 원본 문서 구조 개선, 고화질 이미지 사용 |
| 내용 누락 | 문서에 핵심 내용 명확히 표시 |
| 다운로드 안됨 | 브라우저 캐시 삭제, 시크릿 모드 시도 |
