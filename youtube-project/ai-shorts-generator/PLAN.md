# AI 쇼츠 자동 생성 시스템 기획서

## 목표
개인이 쉽게 AI를 활용해 3D 애니메이션 스타일 쇼츠 영상을 자동으로 생성

## 전체 워크플로우

```
주제 입력 → 스크립트 생성 → 이미지 생성 → 영상 변환 → 음성 추가 → 자막 추가 → 완성
```

## 단계별 상세

### 1. 스크립트 생성 (AI)
- **도구**: ChatGPT API / Claude API
- **입력**: 주제 ("라면만 1년 먹으면?")
- **출력**: 30-60초 분량 스크립트
- **비용**: $0.01 ~ $0.05 / 스크립트

**예시 출력:**
```
씬1: 주인공이 라면을 먹기 시작
씬2: 1개월 후 - 얼굴이 부어오름
씬3: 6개월 후 - 영양 불균형 증상
씬4: 1년 후 - 결과
```

### 2. 이미지 생성 (AI)
- **도구**: Leonardo.ai / Midjourney / DALL-E 3
- **입력**: 씬별 프롬프트
- **출력**: 고퀄리티 이미지 (1080x1920)
- **비용**: 무료 크레딧 사용 가능

**추천**: Leonardo.ai
- 무료: 150 크레딧/일
- 이미지당 5-10 크레딧
- 1일 15-30개 이미지 생성 가능

### 3. 이미지 → 영상 변환
- **도구**: Pika Labs / Runway Gen-3 / Stable Video Diffusion
- **입력**: 정적 이미지
- **출력**: 3-5초 애니메이션 클립
- **비용**: 무료 플랜 있음

**추천**: Pika Labs
- 무료: 월 250 크레딧
- 영상당 10 크레딧
- 월 25개 영상 생성 가능

### 4. 음성 생성 (AI TTS)
- **도구**: ElevenLabs / OpenAI TTS
- **입력**: 스크립트 텍스트
- **출력**: 자연스러운 AI 음성
- **비용**: 무료 10분/월 또는 $5/월

### 5. 영상 편집 & 합성
- **도구**: FFmpeg / MoviePy (Python)
- **기능**:
  - 클립 연결
  - 음성 합성
  - 자막 추가
  - 트랜지션 효과
- **비용**: 무료 (오픈소스)

### 6. 자막 생성
- **도구**: OpenAI Whisper / 자체 타이밍
- **기능**: 자동 자막 생성 및 배치
- **비용**: 무료

## 기술 스택

### 언어
- Python 3.9+

### 주요 라이브러리
- `openai` - GPT API
- `requests` - HTTP 요청
- `moviepy` - 영상 편집
- `pillow` - 이미지 처리
- `elevenlabs` - 음성 생성
- `ffmpeg-python` - 영상 처리

### API 서비스
1. **OpenAI GPT-4** - 스크립트 생성
2. **Leonardo.ai** - 이미지 생성
3. **Pika Labs** - 영상 변환
4. **ElevenLabs** - 음성 생성

## 실행 방법

### 기본 사용
```bash
python main.py --topic "라면만 1년 먹으면?"
```

### 옵션
```bash
python main.py \
  --topic "라면만 1년 먹으면?" \
  --duration 30 \
  --style "3d_cartoon" \
  --voice "korean_male"
```

### 배치 생성
```bash
python batch_generate.py --topics topics.txt
```

## 확장 가능성

### 1단계 (현재)
- 단순 스크립트 → 영상 자동화

### 2단계 (향후)
- 트렌드 분석 → 자동 주제 선정
- 유튜브 자동 업로드
- 썸네일 자동 생성

### 3단계 (고급)
- 조회수 분석 → 최적화
- A/B 테스팅
- 멀티 채널 관리
