# 랭킹 쇼츠 자동 생성기 - 프로젝트 요약

## 📌 프로젝트 개요

**유튜브 쇼츠의 인기 트렌드인 "랭킹 영상"을 자동으로 생성하는 시스템**

예시: "Top 5 Football Goals", "Best TikTok Moments Ranking"

---

## 🎯 프로젝트 목표

### 핵심 목표
TikTok에서 인기 영상을 자동으로 수집하고, 사용자가 선택한 5~7개 영상을 랭킹 형식의 쇼츠로 자동 편집

### 기대 효과
- ⏱️ **시간 절약**: 수작업 대비 70% 이상 작업 시간 단축
- 🎨 **품질 일관성**: 자동화된 편집으로 일관된 품질 유지
- 📈 **생산성 향상**: 하루에 여러 개의 쇼츠 제작 가능

---

## 💡 어떻게 동작하나요?

### 사용자 플로우 (반자동 방식)

```
1. 키워드 입력
   → "football skills" 검색

2. 영상 선택
   → 20~30개 후보 중 5~7개 선택
   → 드래그 앤 드롭으로 랭킹 순서 조정

3. 자동 생성
   → 다운로드 → 편집 → 랭킹 텍스트 추가 → 배경음악

4. 검수 & 다운로드
   → 미리보기 → 승인 → 다운로드
```

**소요 시간**: 약 5~10분

---

## 🔑 핵심 기능

### 1. TikTok 영상 검색
- 해시태그/키워드로 인기 영상 자동 수집
- 조회수, 좋아요 수 기준 필터링
- 20~30개 영상 메타데이터 표시

### 2. 웹 UI로 영상 선택
- 썸네일, 통계 정보 확인
- 체크박스로 5~7개 선택
- 드래그 앤 드롭으로 순서 조정

### 3. 자동 영상 편집
- **크롭 & 리사이즈**: 9:16 세로 포맷 (1080x1920)
- **랭킹 텍스트**: 🥇 #1, 🥈 #2, 🥉 #3...
- **영상 합치기**: 순서대로 이어붙이기
- **배경음악**: 자동 추가 및 볼륨 조절

### 4. 검수 프로세스
- 완성된 영상 미리보기
- 승인/재작업/삭제 선택
- 승인 시 라이브러리 저장

## 🛠️ 기술 스택

### Frontend
- **React.js** + Vite
- **Tailwind CSS** (스타일링)
- **Zustand** (상태 관리)
- **Socket.IO** (실시간 진행 상황)

### Backend
- **FastAPI** (Python 웹 프레임워크)
- **Celery** (비동기 작업 처리)
- **Redis** (메시지 브로커)
- **SQLite** (데이터베이스)

### 영상 처리
- **FFmpeg** (영상 편집 엔진)
- **MoviePy** (Python 영상 처리)
- **yt-dlp** (영상 다운로드)

### 스크래핑
- **TikTokApi** (비공식 Python 라이브러리)
- **Playwright** (브라우저 자동화)

## ⚠️ 주요 도전 과제

### 1. TikTok 스크래핑
**문제**: TikTok이 스크래핑을 막으려고 계속 업데이트
**해결**:
- TikTokApi 라이브러리 사용 (1순위)
- Playwright 커스텀 스크래핑 (백업)
- 수동 URL 입력 (최후 수단)

### 2. 영상 처리 성능
**문제**: FFmpeg 처리 시간 소요
**해결**:
- Celery로 백그라운드 처리
- GPU 가속 (NVENC) 옵션
- 병렬 다운로드

### 3. 저작권 문제
**주의**: 다운로드한 영상의 2차 사용 시 저작권 문제
**대응**:
- 사용자에게 저작권 책임 고지
- 출처 표기 기능 추가 권장
