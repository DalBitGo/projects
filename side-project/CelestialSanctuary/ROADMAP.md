## 🚀 출시 준비 체크리스트

### 기술적 준비
- [ ] **크래시 모니터링**: Firebase Crashlytics 연동
- [ ] **분석 도구**: Firebase Analytics / Amplitude
- [ ] **성능 최적화**: 메모리 누수 점검, 프레임 드롭 방지
- [ ] **ProGuard/R8**: 코드 난독화 설정
- [ ] **앱 서명**: Release 키스토어 생성
- [ ] **버전 관리**: Semantic Versioning (1.0.0)

### 스토어 준비 (Google Play)
- [ ] **앱 아이콘**: 512x512 (고해상도)
- [ ] **피처 그래픽**: 1024x500
- [ ] **스크린샷**: 최소 2장 (폰), 태블릿 권장
- [ ] **앱 설명**: 짧은 설명 (80자), 긴 설명 (4000자)
- [ ] **카테고리**: 라이프스타일 / 엔터테인먼트
- [ ] **콘텐츠 등급**: 설문 완료
- [ ] **개인정보처리방침**: URL 필요

### 법적/정책 준비
- [ ] **개인정보처리방침** 페이지 생성
- [ ] **이용약관** 작성
- [ ] **연령 제한** 검토 (점술 콘텐츠)
- [ ] **광고 정책** 검토 (추후 광고 도입 시)
